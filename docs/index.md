## 這個庫怎麼用

- **打完(或看完)一場比賽** → 由 Claude 讀 editorial 產 digest 草稿(套收錄判準過濾),審閱後入庫;側欄/索引/複習佇列全自動掛載。
- **量化準備** → [checklist](/quant/) 追蹤覆蓋,[艱深題庫](/quant/)面試前快掃;每題「題目 + 技巧 + 答案」。
- **平時複習** → 看[複習佇列](/review/)挑到期的重做,或按 [tag 索引](/cp/tags)集中複習同一招。

> 收錄判準:各難度都收**主流、可遷移**的;偏門 gimmick 不收;~2700 為上限,經典模板工具破例。實例見 [ARC 223 digest](/cp/contests/2026-06-28-ac-arc223)。
