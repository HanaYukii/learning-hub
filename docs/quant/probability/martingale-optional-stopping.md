# 鞅與選擇停時定理(OST) — items

- **考點**:用鞅 + OST 秒算期望步數 / 吸收機率(賭徒破產、首次出現模式、打到邊界)。
- **鞅**:$E[M_{n+1}\mid\mathcal F_n]=M_n$(公平賭局)。
- **OST**:停時 $\tau$ 滿足「有界」或「$E[\tau]<\infty$ 且增量有界」或「一致可積」其一 → $E[M_\tau]=E[M_0]$。

## 對稱 ±1 漫步 $S_n$($p=q=\tfrac12$)
- $S_n$ 是鞅 → 解**吸收機率**。
- $S_n^2-n$ 是鞅 → 解**期望時間**($E[S_\tau^2]=E[\tau]$)。

## 必背結果(從 $k$ 出發,邊界 $0,N$)
- 打到 $N$ 機率 $=k/N$。
- 期望步數 $=k(N-k)$。
- **非對稱**($p\ne q$):指數鞅 $(q/p)^{S_n}$ → 打到 $N$ 機率 $=\dfrac{(q/p)^k-1}{(q/p)^N-1}$;期望時間用鞅 $S_n-(p-q)n$。

## 擲幣首次出現模式(Conway / 賭場論證)
- 期望次數 $=\sum 2^{(\text{自重疊長度})}$。
- HTH $=2^3+2^1=10$;HHH $=2^3+2^2+2^1=14$;HTT $=2^3=8$。**自重疊越多等越久。**

## 易錯
- OST 條件**一定要檢查**(無界停時亂套會錯;經典反例:無邊界對稱漫步首達 $+1$,$E[\tau]=\infty$)。
- 對稱用 $S_n^2-n$;**非對稱期望時間別套對稱公式**,改用 $S_n-(p-q)n$。
- 分清「吸收機率(用 $S_n$ / 指數鞅)」vs「期望時間(用 $S_n^2-n$)」——不同鞅。

## 延伸
- Wald:$E[S_\tau]=E[\tau]\,E[X_1]$。連續版:$W_t$、$W_t^2-t$ 也是鞅(布朗運動首達時間同框架)。
