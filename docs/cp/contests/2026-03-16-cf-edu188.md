# Educational Codeforces Round 188 — 學習重點

> 難度 `~` 為 CF rating(部分為題位估計)。

### [A. Passing the Ball](https://codeforces.com/contest/2204/problem/A)
`模擬` `~800`

::: info 題意
給定 $n$ 個學生排成一列與字串 $s$($s_1=$ `R`、$s_n=$ `L`):第 $i$ 人拿到球後,依 $s_i$ 傳給左鄰($i-1$)或右鄰($i+1$)。

球從 1 號開始,恰好傳 $n$ 次。**求**:至少拿過一次球的學生數。

例:`s = RRLL` → 球走 $1\to2\to3\to2\to3$,共 $3$ 人拿過球。
:::

::: tip 作法
球一路向右,撞上第一個 `RL` 交界後永遠在那兩人間震盪($n$ 次必夠走到)→ 答案 = 第一個 `RL` 中 `L` 的 1-indexed 位置,即 `s.find("RL") + 2`。
:::

### [B. Right Maximum](https://codeforces.com/contest/2204/problem/B)
`貪心` `~1000`

::: info 題意
給定長 $n$ 的整數陣列 $a$。當陣列非空,重複操作:選**最右邊的最大值**,把它連同其後所有元素一起刪除。

**求**:陣列清空前共執行幾次操作。

例:`a = [3, 1, 3, 2]` → 先刪位置 3 的 $3$ 及其後,剩 `[3, 1]`;再刪位置 1 的 $3$ 及其後 → 空,共 $2$ 次。
:::

::: tip 作法
每次切點恰是「最後一個 $a_i \ge$ 前綴 max 的位置」→ 答案 = 滿足 $a_i \ge \max(a_1,\dots,a_{i-1})$ 的位置數,一次掃描維護 max、`a[i] >= mx` 就計數。
:::

### [C. Spring](https://codeforces.com/contest/2204/problem/C)
`數學` `容斥` `~1300`

::: info 題意
Alice、Bob、Carol 分別每 $a$、$b$、$c$ 天造訪水泉一次(第 $a, 2a, \dots$ 天)。當天只有 1 人到,取 $6$ 升;2 人平分各取 $3$ 升;3 人各取 $2$ 升。

**輸出**:前 $m$ 天內三人各自取得的總水量。

例:`a=2, b=3, c=6, m=6` → Alice 第 2、4 天獨取各 $6$、第 6 天三人同至取 $2$,共 $14$;Bob 得 $6+2=8$;Carol 得 $2$。
:::

::: tip 作法
同至天數 = $\lfloor m/\mathrm{lcm}\rfloor$,對每人容斥拆三類天數:獨自 $=\frac ma-\frac m{\mathrm{lcm}(a,b)}-\frac m{\mathrm{lcm}(a,c)}+\frac m{\mathrm{lcm}(a,b,c)}$、恰兩人 $=$ 兩兩重疊和 $-\,2\times$ 三重、三人 $=\frac m{\mathrm{lcm}(a,b,c)}$(除法皆下取整),再按 $6/3/2$ 加權相加。
:::

### [D. Alternating Path](https://codeforces.com/contest/2204/problem/D)
`圖論` `構造` `~1600`

::: info 題意
給定 $n$ 點 $m$ 邊簡單無向圖,你要為每條邊定向。定向後,點列 $v_1,\dots,v_k$(可重複、可任意長)是交錯路徑 ⇔ 邊方向依序為 $v_1\to v_2$、$v_3\to v_2$、$v_3\to v_4$、$v_5\to v_4,\dots$(出、入交替)。

點 $v$ 美 ⇔ 原圖中所有從 $v$ 出發的路徑(不必簡單)定向後都交錯。**求**:美點數的最大值。

例:路徑圖 $1\text{–}2\text{–}3$,定向 $1\to2$、$3\to2$ → $1,3$ 皆美,答案 $2$;三角形無論怎麼定向答案 $0$。
:::

::: tip 作法
長度 1、2 的路徑分別逼出「美點的邊全出」「鄰點其餘邊全入」,遞推即 2-染色定向「A 側 → B 側」:**A 側全美、B 側全不美** → 每個二分圖分量貢獻 $\max(|A|,|B|)$,含奇環分量貢獻 $0$;BFS 染色即可。
:::

### [E. Sum of Digits (and Again)](https://codeforces.com/contest/2204/problem/E)
`構造` `數學` `~2000`

::: info 題意
對正整數 $x$ 定義字串 $S(x)$:不斷把 $x$ 的十進位表示接到右邊;若 $x\le9$ 停止,否則 $x\leftarrow$ 其數位和。如 $S(75)=$ `75123`($75\to12\to3$)、$S(30)=$ `303`、$S(9)=$ `9`。

給定數字字串 $s$,**輸出** $s$ 的任一重排,使其恰為某個 $S(x)$(不得增刪字元)。

例:`s = 12357` → 重排成 `75123` $=S(75)$:取 $y=12$,後綴鏈 `12`+`3`,剩餘數字 $\{7,5\}$ 數位和恰為 $12$。
:::

::: tip 作法
$S(x)=x$ 接 $S(f(x))$($f$ = 數位和),而 $f(x)\le 9|s|$ → **枚舉 $y=f(x)$,整條後綴鏈由 $y$ 唯一決定**;檢查鏈的數字可由 $s$ 扣除、且剩餘數位和恰為 $y$,成立就把剩餘數字由大到小排成 $x$(數位和固定與順序無關,降序自動無前導零)。

```cpp
// sketch:枚舉 y = f(x),後綴鏈由 y 唯一決定
for (int y = 1; y <= 9 * n; y++) {
    string suf = to_string(y);
    for (int v = y; v > 9; suf += to_string(v)) v = digsum(v);  // S(y) 整條鏈
    // 可行 ⇔ suf 各數字次數 ≤ s 的,且 (s 數位總和 − suf 數位總和) == y
    if (feasible(suf, y)) { /* 剩餘數字降序接上 suf,輸出 */ }
}
```

### [F. Sum of Fractions](https://codeforces.com/contest/2204/problem/F)
`貪心` `貢獻` `離線` `~2300`

::: info 題意
定義分數 $\frac xy$ 的一次增大:分子 $+1$,或(當 $y>1$)分母 $-1$(不約分)。$\mathrm{MSF}(b,k)$ = 把陣列 $b$ 變成 $\frac1{b_1},\dots$ 後**恰好**做 $k$ 次增大能得到的最大總和。

給定陣列 $a$ 與 $m$ 個查詢 $k_i$,對每個 $k_i$ **輸出** $\sum_{l\le r}\mathrm{MSF}(a[l..r],\,k_i)\bmod 998244353$。

例:$\mathrm{MSF}([2,3],\,k{=}2)$:把 $\frac12$ 的分母降到 $1$ 再加一次分子 → $\frac21$,總和 $2+\frac13=\frac73$;優於加兩次分子的 $\frac32+\frac13$。
:::

::: tip 作法
單組貪心:全部 $k$ 次都砸在最小分母 $mn$ 的那個分數 —— $k<mn$ 取 $\frac{k+1}{mn}$,否則先降到分母 $1$ 再堆分子得 $k+2-mn$(其餘元素照付 $\frac1{b_j}$;分界 $k=mn-1$ 處兩式同值 $1$)。

全子陣列求和:monotonic stack 算 $a_i$ 當 min 的子陣列數 $cnt_i=(i-L_i)(R_i-i)$,每個 $i$ 的貢獻是**以 $a_i$ 為門檻、關於 $k$ 的分段一次函數** → 掛到排序查詢的差分上一次讀出。

```cpp
// sketch:貢獻 = k 的分段一次函數,掛差分
forn(i, n) {
    long long c = 1ll * (i - L[i]) * (R[i] - i);   // a[i] 當 min(一側取等防重)
    common = (common + (1ll * (i + 1) * (n - i) - c) % MOD * inv(a[i])) % MOD;  // 非 min 照付 1/a_j
    int p = lower_bound(all(k), a[i]) - k.begin();  // 查詢以 a[i] 分界
    long long ci = c % MOD * inv(a[i]) % MOD;
    add(slope, 0, p, ci);       add(inter, 0, p, ci);                                    // (K+1)/a_i
    add(slope, p, m, c % MOD);  add(inter, p, m, c % MOD * (2 - a[i] % MOD + MOD) % MOD); // K+2-a_i
}
// 差分前綴和後:ans[j] = common + slope[j] * k[j] + inter[j](取模)
```

**求**:可能的路徑(集合)數 $\bmod\ mod$。($n$ 大到需矩陣快速冪、$m\le150$、$mod$ 為輸入。)

例:$n=m=2$ → $7$ 種:只走第 1 行 $2$ 種(段 $[1,1]$ 或 $[1,2]$);下到第 2 行時接任一相交段,$[1,1]$ 接 $2$ 種、$[1,2]$ 接 $3$ 種。
:::

::: tip 作法
足跡 ⇔ **段鏈**:每行足跡是連續段、相鄰行段必相交、首段含第 1 欄(進入該行後掃完整段、再從交集處下行,任何段鏈都走得出來)→ 計數段鏈。

直接以段為狀態有 $O(m^2)$ 維;關鍵:**取前綴和後轉移封閉** —— 不相交 ⇔ 全在左或全在右,$dp_{i,l,r}=1+pre_{i-1,m}-pre_{i-1,l-1}-suf_{i-1,r+1}$($+1$ = 鏈從本行開始,配 180° 對稱把「首段含第 1 欄」換成「末段右端 $=m$」),且水平鏡像給 $suf_{i,j}=pre_{i,m-j+1}$ → 狀態壓到 $m+1$ 維(官方 code 走 $2m+1$ 維,本質相同),矩陣快速冪跑 $n$ 行,答案 $pre_{n,m}-pre_{n,m-1}$。

$mod$ 任意 → 轉移全程只用加減乘;$300^3\log n$ 很緊,官方乘法用「**轉置 + u64 累加、每 16 項取一次模**」壓常數($16$ 個 $<10^{18}$ 的積剛好不爆 $2^{64}$)。

```cpp
// sketch:任意模數矩陣乘的常數壓法(官方 code)
mat mul(const mat& a, const mat& b) {
    mat bt = transpose(b), c(k, vector<int>(k));   // 轉置 → 內層連續存取
    forn(x, k) forn(y, k) {
        unsigned long long acc = 0;
        forn(z, k) {
            acc += 1ull * a[x][z] * bt[y][z];
            if ((z & 15) == 15) acc %= mod;        // 每 16 項取一次模
        }
        c[x][y] = acc % mod;
    }
    return c;
}
```
