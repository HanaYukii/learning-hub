# AtCoder Beginner Contest 464 (Japanese Student Championship 2026 Advance Qual) — 學習重點

> 懶人包:每題「題意 + 核心作法」。難度 `~` 為 **CF-equivalent**(kenkoooo difficulty +400 粗估),據官方 editorial 整理。

- **[A. Decisive Battle](https://atcoder.jp/contests/abc464/tasks/abc464_a)** `模擬` `~400`
  題意:給定字串 S(長度 1–99 且為奇數,僅含字元 `E`、`W`),`E` 的個數是東軍士兵數、`W` 的個數是西軍士兵數。
  輸出:東軍較多輸出 `East`、西軍較多輸出 `West`;長度為奇數保證兩軍人數必不相等。
  作法:計數比較即可。

- **[B. Crop](https://atcoder.jp/contests/abc464/tasks/abc464_b)** `模擬` `~500`
  題意:給定 H×W(H, W ≤ 50)黑白影像的字元矩陣,`.` 為白、`#` 為黑,保證至少一個黑格。
  依「上、下、左、右」的順序,反覆刪除位於該側邊緣且整列/整行全白的列與行,直到四邊都刪不動。
  輸出:處理後剩餘的影像(逐列輸出字元矩陣)。
  作法:等價於**所有黑格的 bounding box**——掃出黑格 min/max 行列,直接輸出該子矩形,不必逐次刪。

- **[C. Plumage Palette](https://atcoder.jp/contests/abc464/tasks/abc464_c)** `差分` `~800`
  題意:給定 N 隻鳥與 M 天觀察(N, M ≤ 3×10^5),每隻鳥 i 以三元組 (A_i, D_i, B_i) 描述:第 D_i − 1 天(含)以前是顏色 A_i、自第 D_i 天(含)起是顏色 B_i(顏色編號 1…N,允許 A_i = B_i 即不變色)。
  輸出:M 行,第 j 行為第 j 天全體 N 隻鳥身上相異顏色的數量。
  作法:每隻鳥拆成三個事件(day 1:A 色 +1;day D_i:A 色 −1、B 色 +1)按天 bucket;**維護每色 cnt 與相異數 kind,0→1 時 kind++、1→0 時 kind--**,逐天輸出,O(N+M)。

- **[D. Celester](https://atcoder.jp/contests/abc464/tasks/abc464_d)** `dp` `狀壓` `~1200`
  題意:給定長度 N 的天氣字串 S(`S` 晴、`R` 雨)與整數列 X_1…X_N、Y_1…Y_{N−1}(皆 ≤ 10^9):可任選若干天翻轉其天氣(晴↔雨),翻第 i 天花費 X_i;最終每個「第 i 天雨且第 i+1 天晴」的位置獲得 Y_i。
  輸出:操作後可達到的最大淨快樂值(ΣY 收益 − ΣX 花費);多測,ΣN ≤ 2×10^5。
  作法:相鄰貢獻只取決於連續兩天 → **只保留「前一天=晴」「前一天=雨」兩狀態**的滾動 DP,四種轉移各扣翻轉 cost,O(N)。

- **[E. Fill-Rect Query](https://atcoder.jp/contests/abc464/tasks/abc464_e)** `差分` `離線` `~1500`
  題意:給定 H×W 網格(H×W ≤ 10^6),初始每格都寫著字母 `A`。依序執行 Q ≤ 2×10^5 次操作,第 i 次給 (R_i, C_i, X_i):把左上角 (1,1)、右下角 (R_i, C_i) 的矩形內所有格子覆寫成大寫字母 X_i。
  輸出:所有操作結束後的盤面,共 H 行、每行 W 個字母。
  作法:**用操作序號 i(而非字母)覆寫:每格答案 = 滿足 R_i≥r、C_i≥c 的最大 i**——把 i 放到 (R_i, C_i),從右下往左上做二維後綴 max,最後把 i 還原成 X_i。另解:逆序處理、只填尚未寫過的格子,均攤 O(HW+Q)。

- **[F. Random Vault Heist](https://atcoder.jp/contests/abc464/tasks/abc464_f)** `機率期望` `MITM` `組合計數` `~2600`
  題意:給定 N ≤ 40 個保險箱的金額 A_1…A_N(A_i ≤ 10^16)與門檻 X(1 ≤ X ≤ ΣA_i)。搶匪反覆執行「從尚未開過的箱子中等機率隨機選一個,打開並拿走裡面全部的錢」,直到偷得總額 ≥ X 為止停手。
  輸出:偷得總額的期望值 mod 998244353(有理數取模)。
  作法:**「隨機開到停」改看成「隨機排成一列、取前綴」**:途中經過集合 S 的機率 = 1/C(N,|S|),答案 = Σ_{ΣA(S)<X} (剩餘平均金額)/C(N,|S|);同 |S|=k 分母相同,只需每個 k 的「合法集合數 + ΣA 總和」→ **折半枚舉**:右半按 (大小, 和) 排序 + prefix sum,左半枚舉後二分/雙指針合併,O(N·2^{N/2})。

- **[G. Celester 2](https://atcoder.jp/contests/abc464/tasks/abc464_g)** `構造` `反悔貪心` `~3000`
  題意:給定長度 N(ΣN ≤ 10^6)的天氣字串 S(`S` 晴、`R` 雨),一次操作可把某一天的天氣晴↔雨互換。對每個 k = 0, 1, …, N 問:至多操作 k 次後,「第 i 天雨且第 i+1 天晴」的位置對數最大值。
  輸出:一行 N+1 個整數 A_0 A_1 … A_N(A_k 為至多 k 次操作的答案);多測。
  作法:**前後補 S…R,轉成相鄰異同的 01 列;改一字 = 翻相鄰兩位,答案 +1 等價於「把兩個 0 配對」,配對距離 D 的一對花 D 次翻轉**。歸約成:對每個 j 求「相鄰 0 距離陣列 D 中選 j 個、不可選相鄰、和最小」——正是 JOI Candies 反悔貪心(選 D_i 後把 D_{i-1}, D_i, D_{i+1} 併成 D_{i-1}+D_{i+1}−D_i),priority_queue + 雙向鏈表 O(N log N)(cost 和 ≤ N,桶排可做到線性)。
