### [A. Unusual-Constraint Knapsack](https://atcoder.jp/contests/arc223/tasks/arc223_a)
`貪心` `背包` `AtC~1000`

::: info 題意
給定 $N$ 件物品(第 $i$ 件有重量 $w_i$、價值 $v_i$)與背包容量 $W$,保證重量超遞增:對每個 $i\ge2$ 有 $\sum_{j<i} w_j < w_i$。

從中選取一個子集,總重量不得超過 $W$。輸出:可達的最大總價值。

範圍:$N\le60$、$w_i\le W\le10^{18}$、$v_i\le10^{16}$,多測 $\sum N\le2\times10^5$。

例:$w=(1,2,4)$、$v=(3,1,3)$、$W=5$ → 取第 $1$、$3$ 件(總重 $5\le W$)得 $3+3=6$;窮舉全部 $8$ 個子集無更優,答案 $6$。
:::

::: tip 作法
$N\le60$、值域 $10^{18}$,一般背包 DP 開不了,錢在「超遞增」上。從最重的第 $N$ 件往下決策、維護剩餘容量 $R$:

- 放不下($w_i>R$)只能跳過;
- 放得下時,若**選擇跳過**,那超遞增給了 $\sum_{j<i}w_j<w_i\le R$——**前 $i-1$ 件全部塞得進**,這個分支的最優值直接就是前綴價值和,O(1) 封頂,不必往下搜;
- 只有**選擇拿**的分支真正遞迴下去($R\leftarrow R-w_i$)。

於是搜尋樹退化成一條鏈:每層一次 O(1) 的 skip 估值 + 一條 take 路徑,整體 $O(N)$。超遞增序列的固定套路——「skip 的後續是平凡的」。

```cpp
// sketch:「不取」分支 O(1) 終止,只有「取」往下遞迴 → O(N)
long long solve(int i, long long R) {          // 前 i 件、剩餘容量 R
    if (i == 0) return 0;
    if (w[i] > R) return solve(i - 1, R);      // 放不下:只能跳過
    long long skip = pre_v[i - 1];             // 不取:∑w < w_i ≤ R,前 i-1 件全取
    long long take = v[i] + solve(i - 1, R - w[i]);
    return max(skip, take);                    // 兩分支取大
}
```
:::

### [B. Valid Arrays by K-Divisible Swaps](https://atcoder.jp/contests/arc223/tasks/arc223_b)
`組合計數` `AtC~1300`

::: info 題意
給定長度 $N$ 的整數序列 $A$($1\le A_i\le10^9$)與整數 $K\le10^9$。

每次操作可選相鄰兩項,若 $A_i+A_{i+1}$ 是 $K$ 的倍數則交換之;操作可執行任意次(含零次)。

輸出:從初始序列出發可達的不同序列總數,mod 998244353。範圍:$N\le2\times10^5$,多測 $\sum N\le2\times10^5$。

例:$K=3$、$A=(1,2,1)$ → 兩處相鄰和($1{+}2$、$2{+}1$)皆為 $3$ 的倍數可交換:$(1,2,1)\to(2,1,1)$ 或 $(1,1,2)$,再操作只會退回,共 $3$ 種。
:::

::: tip 作法
交換條件只看 mod $K$:能換的相鄰對是餘數互補的 $(r, K-r)$。而元素本身不會變,只會移動——所以整個過程中,每個元素的餘數類是固定標籤。

先切段:相鄰兩數若「和、差都不是 $K$ 的倍數」(既非互補、亦非同類),這兩個位置之間永遠換不過去,從這裡切斷後各段獨立、方法數相乘。段內只剩兩種情形:

- **$2r\equiv0\pmod K$(即 $r=0$ 或 $K/2$)**:同類之間也互補,任兩相鄰都能換 → 段內可自由排列;
- 否則段內恰含 $r$、$K-r$ 兩類:只有異類相鄰能換(= 異類間的相鄰對換),**同類元素的相對順序永遠不變**,可達序列 = 兩條固定順序的序列的所有交錯法,方法數 = $\binom{\text{段長}}{r\ \text{類個數}}$。

O(N)。「交換條件 → 等價類 → 段內自由度」是這型可達性計數的標準拆法。
:::

### [C. Whole Product of Pairwise Distances](https://atcoder.jp/contests/arc223/tasks/arc223_c)
`數論` `逆序對` `AtC~1700`

::: info 題意
給定 $N$ 個正整數 $A_1,\dots,A_N$($1\le A_i\le10^9$)。

輸出:全體點對絕對差之積 $\prod_{1\le i<j\le N}|A_i-A_j|$ 除以 $N$ 的餘數——模數是序列長度 $N$ 本身而非質數。

範圍:$2\le N\le2\times10^5$,多測 $\sum N\le2\times10^5$。

例:$N=3$、$A=(1,5,3)$ → $|1{-}5|\cdot|1{-}3|\cdot|5{-}3|=4\cdot2\cdot2=16$,除以 $3$ 餘 $1$,輸出 $1$。
:::

::: tip 作法
模數不是質數、還恰好等於元素個數 $N$——這個「巧合」就是題眼。先排序(絕對值就都變成正差)、令 $B_i=A_i\bmod N$。**鴿籠**:$N$ 個數 mod $N$,只要有兩個相同,對應的差 $\equiv0$、整個積 $\equiv0$ 直接輸出;否則 $B$ 恰是 $\{0,\dots,N-1\}$ 的完整剩餘系,是一個排列。

排列的情形套 **Vandermonde 恆等式**:對排列 $\sigma$,$\prod_{i<j}(\sigma_j-\sigma_i)=(-1)^{I}\prod_{i<j}(j-i)$,其中 $I$ 是逆序數(把 $\sigma$ 冒泡排回升序,每次相鄰對換讓積變號一次)。右邊的 $\prod_{i<j}(j-i)$ 按差值 $d=j-i$ 分組,恰為 $\prod_{d=1}^{N-1}d^{\,N-d}$,直接可算。逆序數 BIT / 歸併 $O(N\log N)$,整體就這個複雜度。

```cpp
// sketch:排序 → mod N → 檢重複 → 逆序數 + ∏ d^{N-d}
sort(a.begin(), a.end());
for (auto &x : a) x %= n;                    // B_i = A_i mod N(排序後)
if (has_duplicate(a)) { print(0); return; }  // B 非排列 → 積 ≡ 0
long long I = count_inversions(a);           // BIT / 歸併,O(N log N)
long long ans = 1;
for (long long d = 1; d < n; ++d)
    ans = ans * pow_mod(d, n - d, n) % n;    // ∏_{i<j}(j-i) = ∏ d^{N-d}
if (I & 1) ans = (n - ans) % n;              // 乘上 (-1)^I
```
:::

### [D. Xpectation of Cards in Hand with Laboratory](https://atcoder.jp/contests/arc223/tasks/arc223_d)
`機率期望` `組合計數` `反射原理` `AtC~2300`

::: info 題意
給定整數 $A,B,K$:$A$ 張 draw 卡與 $B$ 張普通卡均勻隨機洗成一疊牌堆,先從堆頂抽 $K$ 張入手。

接著只要手中還有 draw 卡就重複:棄掉一張 draw 卡,並從堆頂補 $\min(c,2)$ 張($c$ 為牌堆剩餘張數),直到手中沒有 draw 卡為止。

輸出:結束時手牌張數的期望值,mod 998244353(有理數取模)。範圍:$1\le K\le A+B\le10^7$,多測 $\sum(A+B)\le10^7$。

例:$A=1,B=2,K=1$ → 牌堆 $3$ 種排法各 $1/3$:draw 卡在頂時首抽得之,棄掉再補 $2$ 張普通卡,終手 $2$ 張;draw 卡在第 $2$、$3$ 位時首抽是普通卡、直接停,終手 $1$ 張。期望 $=\frac{2+1+1}{3}=\frac43$。
:::

::: tip 作法
過程看起來動態,其實只由牌堆的排列決定,先把它翻成**格路**:沿牌堆從頂往下讀,抽出 draw 卡走右步($a{+}1$)、普通卡走上步($b{+}1$)。手牌數帳目:抽了 $a+b$ 張、棄了 $a$ 張 draw,手上就是 $b$ 張普通卡;而「手中 draw 恰好用罄」翻成路徑語言是**首次碰到直線 $b=a+K$**(抽 $K$ 張起手 + 每棄一補二,推平衡式即得)。所以終止只有兩種:首次碰線(終手 $b=a+K$),或一路沒碰線直到牌堆見底(把 draw 棄完,終手 $B$)。

碰線點沿著一條直線分佈,只有 $O(A+B)$ 個——對每個碰線點算機率再加權即可。**「首次碰線」是 first-passage 計數,反射原理出場**:走到 $(a, a+K)$ 且首次碰線的路徑數 $=\binom{a+b-1}{a}-\binom{a+b-1}{a-1}$(ballot 式),乘上線後剩牌的自由排法 $\binom{A-a+B-b}{A-a}$;「始終不碰線」的總數也有閉式 $\binom{A+B}{A}-\binom{A+B}{A+K}$(全部減碰過線的,反射一次)。各乘終手牌數、除以 $\binom{A+B}{A}$,$O(A+B)$。

### [E. Yin-Yang Two Bits Insertion](https://atcoder.jp/contests/arc223/tasks/arc223_e)
`差分` `不變量` `dp` `AtC~2550`

::: info 題意
給定 0/1 序列 $A$(長 $N$)與 $B$(長 $M$)。

每次操作選位置 $i$,在 $A_i$ 與 $A_{i+1}$ 之間依序插入 $1-A_i$、$1-A_{i+1}$ 兩個元素(如 $(0,1,1)$ 取 $i=2$ 變 $(0,1,0,0,1)$),可執行任意次。

輸出:能否把 $A$ 變成 $B$,能則 `Yes` 否則 `No`。範圍:$2\le N\le M\le2\times10^5$,多測 $\sum(N+M)\le4\times10^5$。

例:$A=(0,1)$ → 唯一的一步 $i=1$ 得 $(0,1,0,1)$,故 $B=(0,1,0,1)$ 輸出 `Yes`;而 $B=(0,1,1,1)$ 長度同為 $4$ 卻到不了(差分中 $0$ 的個數 $2\ne0$),輸出 `No`。
:::

::: tip 作法
插入型操作直接分析很亂,先**取差分** $A'_i=[A_i\ne A_{i+1}]$ 換座標:原操作把 $(x,y)$ 之間插進 $(1{-}x,\,1{-}y)$,對差分逐位驗一下——$x\oplus(1{-}x)=1$、$(1{-}x)\oplus(1{-}y)=x\oplus y$、$(1{-}y)\oplus y=1$——所以差分位 $d$ 的變化就是 $d\to 1\,d\,1$,即 **$0\to101$、$1\to111$**。兩個不變量立刻現形:原串首尾字元(插入不動兩端)、以及**差分中 $0$ 的個數**($d\to1d1$ 原封保留 $d$)。

第二層換座標:以差分的 $0$ 為隔板分段、記各段 $1$ 的個數。$0\to101$ 是在某個 $0$ 兩側各補一個 $1$,即「**相鄰兩段各 $+1$**」;$1\to111$ 是「**某非空段 $+2$**」。判可達變成:能否用這兩種加法把 $A$ 的段向量變成 $B$ 的。搜尋空間看似無限,但有一個替換論證封頂:**同一對相鄰段做「各 +1」$\ge3$ 次,其中兩次可以改用兩個「+2」達成同樣效果**——所以每對至多做 2 次。剩下的就是左到右 DP(狀態 = 前一對的操作次數 0/1/2),$O(|A|+|B|)$。

兩步變換各自都不難,串起來才是這題的難度:**插入題先差分,差分後再找隔板結構**。
:::

### [F. Zonal Score Maximization](https://atcoder.jp/contests/arc223/tasks/arc223_f)
`組合計數` `NTT` `矩陣` `AtC~3150`

::: info 題意
對 $(1,\dots,N)$ 的排列 $P$,定義 $f(P)$ 為:把 $P$ 切成若干段長 $\ge2$ 的連續段,各段 $(\max+\min)$ 總和在所有合法切法中的最大值。

給定序列 $Q$(每項為 $-1$ 或 $1\sim N$ 中的整數,非 $-1$ 者互異)與目標值 $X\le10^{18}$。

輸出:滿足「$Q_i\ne-1$ 處 $P_i=Q_i$」且 $f(P)=X$ 的排列 $P$ 個數,mod 998244353。範圍:$N\le10^5$,多測 $\sum N\le10^5$。

例:$N=3$、$Q=(-1,-1,-1)$ → 長 $3$ 只能整段不切,$f(P)=3+1=4$ 對全部 $3!=6$ 個排列同值;故 $X=4$ 時答案 $6$,其餘 $X$ 答案 $0$。
:::

::: tip 作法
先把 $f$ 算清楚。任何長 $\ge2$ 的段都有 $\max+\min\le$ 段內和,且**等號恰在段長 $=2$ 時成立**——所以 $N$ 偶時全切成長 2 的段,$f(P)=\frac{N(N+1)}{2}$ 對每個排列都一樣:$X$ 等於它時答案是 $c!$($c$ 為 $-1$ 的個數),否則 0。

$N$ 奇時必有恰一段長 3(其餘全長 2),長 3 的段損失其中位數:**$f(P)=\frac{N(N+1)}{2}-\min\operatorname{median}(P_i,P_{i+1},P_{i+2})$**,min 取遍奇數起點 $i$(段長配置逼出的奇偶約束)。令 $p=\frac{N(N+1)}{2}-X$,問題變成數「min median $=p$」的排列,差分成「$\ge p$ 的個數 − $\ge p+1$ 的個數」。

數「所有窗口 median $\ge p$」:把值以 $<p$ / $\ge p$ 二值化成 $B$,條件化為每個奇起點窗口 $B_{i}+B_{i+1}+B_{i+2}\ge2$——這是 01 序列上的線性掃描約束,用 $2\times2$ 的**形式冪級數轉移矩陣**(逐位置乘、變數記錄用掉幾個 0)**分治連乘 + NTT 卷積**,取「0 恰用 $p-1$ 個」的係數,$O(N\log^2 N)$;最後乘上自由位置的階乘轉回排列計數。

重的是實作,骨架其實是三個熟面孔的疊加:**max+min 的切段分析 → 對答案差分 → 二值化後矩陣連乘計數**。
:::
