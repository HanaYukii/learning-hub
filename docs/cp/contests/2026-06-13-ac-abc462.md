# AtCoder Beginner Contest 462 (CodeQUEEN 2026 qual) — 學習重點

> 懶人包:每題「題意 + 核心作法」。各難度都收主流可遷移的;偏門/太難略過。難度 `~` 為 CF-equivalent。

A–B 熱身略過(A 取出字串中的數字;B 反轉「誰送給誰」的鄰接表)。C 起逐題列。

- **[C. Not Covered Points](https://atcoder.jp/contests/abc462/tasks/abc462_c)** `Pareto` `~700`  
  題意:N≤3×10^5 個點,X、Y 各為 1..N 的排列;求有幾個 i,使以 (0,0)、(X_i,Y_i) 為對角的開矩形內部不含任何點。  
  作法:點 i 合法 ⟺ 所有 X_j<X_i 的 j 都有 Y_j>Y_i;**依 X 升序排序後即「Y 是前綴嚴格最小值」的點數**,掃一遍維護最小值,O(N log N)。

- **[D. Accomplice](https://atcoder.jp/contests/abc462/tasks/abc462_d)** `差分` `組合計數` `~1200`  
  題意:N≤2×10^5 名嫌犯各於 [S_i,T_i] 在館(T_i≤10^6);兩名共犯須在犯行區間 [x,x+D] 全程在館,求(共犯無序對、開始時刻 x)的組合數。  
  作法:時刻 t 貢獻 C(c_t,2),c_t = 區間 [t,t+D] 全程在館人數。**把每人退館時刻改成 T_i−D(滯留<D 者忽略),c_t 就退化成普通的「時刻 t 在館人數」**;差分/imos 掃出所有 c_t 累加,O(N+M)。

- **[E. Alternating Costs](https://atcoder.jp/contests/abc462/tasks/abc462_e)** `數學` `分類討論` `~1600`  
  題意:棋子從 (0,0) 單位步走到 (X,Y)(|X|,|Y|≤10^9);奇數步橫移費 A、縱移費 B,偶數步相反;T≤2×10^5 筆,求最小總花費。  
  作法:對稱性設 X,Y≥0;X+Y 奇數時退一步 min(f(X−1,Y)+A, f(X,Y−1)+B) 化為偶數。偶數時共 2K 步、奇偶步各 K 次,**A,B 與 X,Y 都可 swap 成 A≤B、X≤Y,最小費用為一次函數 g(K)=2KA+(Y−K)(B−A)**;端點取極值,答案 = min(g((X+Y)/2), g(Y)),每筆 O(1)。

- **[F. More ABC](https://atcoder.jp/contests/abc462/tasks/abc462_f)** `dp` `字串` `~2400`  
  題意:給大寫字串 S(Σ|S|≤3×10^5)與 K≤10,以最少次單字元替換讓子串 ABC 的出現次數恰好增加 K 個;不可能輸出 −1。  
  作法:dp[i][j] = 前綴 S'₁..ᵢ 比原串至少多 j 個 ABC 的最少改字數;**每個位置只分「S'₍ᵢ₋₂..ᵢ₎=ABC」成立與否兩種轉移**:dp[i][j]=min(dp[i−3][j−1+Xᵢ]+Yᵢ, dp[i−1][j+Zᵢ]),X/Y/Z 皆 O(1) 預處理,整體 O(NK)。另解:f(x)=「多出 ≥x 個 ABC 的最小花費」為凸,Alien DP 二分免除額 O(N log N)。

- **[G. Completely Wrong](https://atcoder.jp/contests/abc462/tasks/abc462_g)** `容斥` `組合計數` `多項式` `NTT` `分治` `~2600`  
  題意:N≤2×10^5 顆球,球 i 顏色 C_i;第 k 次等機率抽走一球,若其顏色恰為 G_k 得 1 分;求 N 次後總分為 0 的機率 mod 998244353。  
  作法:抽球順序即隨機排列 P,總分 0 ⟺ 全程 C_{P_k}≠G_k,容斥成 (1/N!)Σ_A (−1)^|A| |∩_{i∈A} Sᵢ|;記 C、G 中值 k 的個數為 X_k,Y_k,固定各 k 被配對的次數後**貢獻按顏色獨立,化成 f_k(x)=Σ_j (−1)ʲ C(X_k,j) C(Y_k,j) j! xʲ,取 f=∏ f_k 則答案 = (1/N!)Σ cᵢ(N−i)!,且 ∏ 總次數 ≤N**;小次數優先合併的分治 NTT,O(N log²N)。
