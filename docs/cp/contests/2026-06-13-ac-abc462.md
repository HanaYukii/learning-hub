# AtCoder Beginner Contest 462 (CodeQUEEN 2026 qual) — 學習重點

> 懶人包:每題「技巧 + 作法」。各難度都收主流可遷移的;偏門/太難略過。難度 `~` 為 CF-equivalent。

A–B 熱身略過(A 取出字串中的數字;B 反轉「誰送給誰」的鄰接表)。C 起逐題列。

- **[C. Not Covered Points](https://atcoder.jp/contests/abc462/tasks/abc462_c)** `Pareto` `~700`
  作法:點 i 合法 ⟺ 對所有 X_j<X_i 都有 Y_j>Y_i。**依 X 升序排序後,等價於「Y 是前綴嚴格最小值」的點數**;掃一遍維護 Y_min 計數,O(N log N)。dominance/staircase 計數的入門手感。

- **[D. Accomplice](https://atcoder.jp/contests/abc462/tasks/abc462_d)** `差分` `組合計數` `~1200`
  作法:犯行開始時刻 t 貢獻 C(c_t, 2),c_t = 區間 [t, t+D] 全程在館人數。**把每人退館時刻改成 T_i−D(滯留<D 者忽略),c_t 就退化成「時刻 t 的在館人數」**;用差分/imos 掃出所有 c_t 再累加組合數,O(N+M)。

- **[E. Alternating Costs](https://atcoder.jp/contests/abc462/tasks/abc462_e)** `數學` `分類討論` `~1600`
  作法:對稱性設 X,Y≥0;X+Y 為奇數時退一步 f=min(f(X−1,Y)+A, f(X,Y−1)+B) 化為偶數。偶數時共 2K 步、奇偶步各半,**可 swap 成 A≤B、X≤Y,總花費是 K 的一次函數 g(K)=2KA+(Y−K)(B−A)**;一次函數在端點取極值,答案 = min(g((X+Y)/2), g(Y)),每筆 O(1)。

- **[F. More ABC](https://atcoder.jp/contests/abc462/tasks/abc462_f)** `dp` `字串` `~2400`
  作法:dp[i][j] = 讓前綴 S'₁..ᵢ 比原串「至少多 j 個」ABC 所需的最少改字數。**每個位置只需分「S'₍ᵢ₋₂..ᵢ₎=ABC」成立與否兩種轉移**:dp[i][j]=min(dp[i−3][j−1+Xᵢ]+Yᵢ, dp[i−1][j+Zᵢ]),其中 Zᵢ/Xᵢ/Yᵢ 都能 O(1) 預處理,整體 O(NK)。另解:f(x)=「改出 ≥x 個 ABC 的最小花費」為凸函數,Alien DP 二分免除額 O(N log N)。

- **[G. Completely Wrong](https://atcoder.jp/contests/abc462/tasks/abc462_g)** `容斥` `組合計數` `多項式` `NTT` `分治` `~2600`
  作法:答案 = (1/N!)|∩ S̄ᵢ|,容斥成 (1/N!)Σ_A (−1)^|A| |∩_{i∈A} Sᵢ|。設 Cᵢ=k、Gᵢ=k 的個數為 X_k, Y_k,固定每個值 k 被配對的次數 D_k;**每個 k 的貢獻獨立化成多項式 f_k(x)=Σ_j (−1)ʲ C(X_k,j) C(Y_k,j) j! xʲ**,取 f=∏ f_k,答案 = (1/N!)Σ c_i (N−i)!。∏ 的總次數 = N,以小次數優先合併的分治NTT,O(N log²N)。
