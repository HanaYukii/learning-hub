### [F. More ABC](https://atcoder.jp/contests/abc462/tasks/abc462_f)
`dp` `字串` `AtC~1950`

::: info 題意
每筆測資給大寫英文字串 S(3≤|S|≤3×10^5,全部測資 Σ|S|≤3×10^5)與整數 K(1≤K≤10)。

一次替換 = 把 S 的任一字元改成任意大寫字母;目標是讓 ABC 作為連續子字串的出現次數恰好比原字串多 K 個。

求:達成目標所需的最少替換字元數;不可能則輸出 −1。

例:`K=1`:`S="ABX"` 改 `X→C` 即多 1 個 ABC,答案 1;`S="ABC"` 長度 3 至多容納 1 個 ABC 而原本已有 1 個 → 答案 −1。
:::

::: tip 作法
dp[i][j] = 前綴 S'₁..ᵢ 比原串至少多 j 個 ABC 的最少改字數;**每個位置只分「S'₍ᵢ₋₂..ᵢ₎=ABC」成立與否兩種轉移**:dp[i][j]=min(dp[i−3][j−1+Xᵢ]+Yᵢ, dp[i−1][j+Zᵢ]),X/Y/Z 皆 O(1) 預處理,整體 O(NK)。

另解:f(x)=「多出 ≥x 個 ABC 的最小花費」為凸,Alien DP 二分免除額 O(N log N)。

```cpp
// sketch:DP 轉移(X/Y/Z 皆由原串 O(1) 預處理)
// dp[i][j] = 前綴 i、比原串至少多 j 個 ABC 的最少改字數
for (int i = 3; i <= n; ++i)
    for (int j = 0; j <= K; ++j)
        dp[i][j] = min(dp[i-3][j-1+X[i]] + Y[i],  // 令 S'[i-2..i]="ABC"
                       dp[i-1][j+Z[i]]);          // S'[i-2..i]≠"ABC"
```
:::

### [G. Completely Wrong](https://atcoder.jp/contests/abc462/tasks/abc462_g)
`容斥` `組合計數` `多項式` `NTT` `分治` `AtC~2250`

::: info 題意
箱中有 N 顆球(N≤2×10^5),球 i 塗顏色 C_i;另給長度 N 的序列 G(1≤C_i,G_k≤N)。

依序做 N 次操作:第 k 次自箱中等機率抽出一球並丟棄,若其顏色恰為 G_k 得 1 分。

求:N 次操作後總得分為 0 的機率,以 mod 998244353 的有理數取模形式輸出。

例:`N=2`,`C=[1,2]`,`G=[2,1]` → 抽序 (球1,球2) 顏色依序 (1,2) 全程不中;抽序 (球2,球1) 第一抽顏色 2 恰為 G_1 → 得分 0 的機率 = 1/2,輸出 499122177。
:::

::: tip 作法
抽球順序即隨機排列 P,總分 0 ⟺ 全程 C_{P_k}≠G_k,容斥成 (1/N!)Σ_A (−1)^|A| |∩_{i∈A} Sᵢ|;

記 C、G 中值 k 的個數為 X_k,Y_k,固定各 k 被配對的次數後**貢獻按顏色獨立,化成 f_k(x)=Σ_j (−1)ʲ C(X_k,j) C(Y_k,j) j! xʲ,取 f=∏ f_k 則答案 = (1/N!)Σ cᵢ(N−i)!,且 ∏ 總次數 ≤N**;

小次數優先合併的分治 NTT,O(N log²N)。
:::
