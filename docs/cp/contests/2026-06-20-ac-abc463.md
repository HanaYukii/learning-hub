# AtCoder Beginner Contest 463 — 學習重點

### [F. Senshuraku](https://atcoder.jp/contests/abc463/tasks/abc463_f)
`機率期望` `組合計數` `AtC~2150`

::: info 題意
給定整數 $N$ 與長度 $2N$ 的整數陣列 $A$,$A_i$ 為第 $i$ 名選手賽前已有的勝場數($1\le N\le 2\times10^5$、$0\le A_i<2N$)。

接下來進行最後 $N$ 場比賽:第 $i$ 場($1\le i\le N$)由第 $2i{-}1$ 與第 $2i$ 名選手對戰,各場勝負獨立、雙方各以機率 $\tfrac12$ 獲勝。全部打完後,從總勝數最多的選手中均勻隨機選出一人為優勝者。

輸出:對每個 $i=1,\dots,2N$,第 $i$ 名選手成為優勝者的機率 $\bmod 998244353$。

例:$N=1$、$A=[1,0]$ → 唯一一場由 1 對 2:1 勝(機率 $\tfrac12$)→ 勝數 $(2,0)$,1 為優勝;2 勝 → $(1,1)$ 平手均勻抽人各半。故 $P_1=\tfrac12+\tfrac14=\tfrac34$、$P_2=\tfrac14$。
:::

::: tip 作法
優勝者總勝數必為 $W$ 或 $W{+}1$($W$=賽前最多勝)→ $\le W{-}2$ 者勝數歸 $0$,末 $N$ 場依雙方勝數 $(W/W{-}1/0)$ 分 6 類。

**枚舉優勝勝數後每場只剩兩型:type-1($\tfrac12$ 出 $1$ 名候選、$\tfrac12$ 出 $0$ 名)與 type-2(機率 $p$ 出 $k$ 名、否則矛盾)**,對每個「候選人數 × 勝數」組合用二項式係數 $O(1)$ 聚合機率求和。
:::

### [G. Random Walk Distance](https://atcoder.jp/contests/abc463/tasks/abc463_g)
`機率期望` `組合計數` `根號` `AtC~2450`

::: info 題意
給定 $T$ 組詢問,每組一個正整數 $N$ 與一個整數 $X$($1\le T,N\le 2\times10^5$、$|X|\le 2\times10^5$)。

每組詢問中,從數線座標 $0$ 出發走 $N$ 步,每步獨立且等機率地移動 $+1$ 或 $-1$,設走完後的終點座標為 $x'$。

輸出:每組詢問一行,期望值 $E[|x'-X|]\bmod 998244353$。

例:$N=2$、$X=1$ → 終點 $x'\in\{-2,0,+2\}$,機率 $\tfrac14,\tfrac12,\tfrac14$;$|x'-1|$ 依序為 $3,1,1$ → $E=\tfrac34+\tfrac12+\tfrac14=\tfrac32$。
:::

::: tip 作法
$P(x'=2i-N)=\binom{N}{i}/2^N$,答案化簡成部分二項和 $f(N,M)=\sum_{i<M}\binom{N}{i}$、$g(N,M)=\sum_{i<M} i\binom{N}{i}$($M=\lceil\tfrac{N+X}{2}\rceil$)。

**寶石:$f,g$ 沒有封閉式,但對 $N$、$M$ 各有 $O(1)$ 遞推**($f(N,M{+}1)-f(N,M)=\binom{N}{M}$、$f(N{+}1,M)=2f(N,M)-\binom{N}{M-1}$)→ 把各詢問 $(N,M)$ 當二維點跑莫隊,$O((N+|X|)\sqrt{T})$。

```cpp
// sketch:詢問 (N, M) 視為二維點跑莫隊,f = Σ_{i<M} C(N,i) 四方向 O(1) 轉移
void incM() { f = (f + C(N, M)) % MOD; ++M; }                      // f(N,M+1) = f + C(N,M)
void decM() { --M; f = (f - C(N, M) + MOD) % MOD; }
void incN() { f = (2 * f % MOD - C(N, M - 1) + MOD) % MOD; ++N; }  // f(N+1,M) = 2f - C(N,M-1)
void decN() { --N; f = (f + C(N, M - 1)) % MOD * inv2 % MOD; }     // 反向:乘 inv2
// g = Σ_{i<M} i·C(N,i) 同樣四方向 O(1) 轉移;詢問按 (N 分塊, M) 排序
for (auto q : qs) {
    while (N < q.N) incN();  while (N > q.N) decN();
    while (M < q.M) incM();  while (M > q.M) decM();
    ans[q.id] = combine(f, g, q);   // 由 f, g 還原 E[|x'-X|]
}
```
:::
