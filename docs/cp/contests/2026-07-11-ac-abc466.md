# AtCoder Beginner Contest 466 — 學習重點

> 懶人包:每題「題意 + 核心作法」。各難度都收主流可遷移的;偏門/太難略過。難度 `~` 為 CF-equivalent。
>
> A–B 熱身略過(A:判斷 `max Xᵢ ≥ 0`;B:逐色取最大 size、初值 −1 免特判)。從 C 開始列。

- **[C. Count Close Pairs](https://atcoder.jp/contests/abc466/tasks/abc466_c)** `互動` `~1100`
  題意:數線上有點 1..N(N≤10³)由左到右排列,只給你 N;每次可問「點 i、j 距離是否 ≤1」,**至多 2N 次**;求距離 ≤1 的點對數。
  作法:距離 ≤1 具單調性——設 Mᵢ = 從 i 起最遠仍 ≤1 的點,則 **M₁≤M₂≤…≤M_N**,可用單調雙指針:L,R 從 1,2 起,問 (L,R),Yes 則 R++,No 則定 M_L=R−1、L++;每步 L+R 至少 +1,總問數 ≤2N−2 卡進預算。答案 = Σ(Mᵢ−i)。

- **[D. Placing Rooks](https://atcoder.jp/contests/abc466/tasks/abc466_d)** `模擬` `~900`
  題意:N×N 空盤做 M 次操作(N,M≤3×10⁵);第 i 次先清空第 Rᵢ 列與第 Cᵢ 行,再於 (Rᵢ,Cᵢ) 放一子;求 M 次後盤上子數。
  作法:一子最終存活 ⟺ 之後沒有任何操作再碰它的行或列。**逆序掃描操作**(M→1),用兩個 bool 陣列記「已被更晚操作用過的行/列」,當前 (Rᵢ,Cᵢ) 行列都還沒被用就計數 +1,再標記。O(N+M)。

- **[E. Range Flip](https://atcoder.jp/contests/abc466/tasks/abc466_e)** `dp` `貪心` `~1400`
  題意:N 張卡(N≤2×10⁵)正面 Aᵢ、反面 Bᵢ,初始全正面朝上;至多做 K 次(K≤10)區間翻轉 [l,r];求最終朝上面數字和最大。
  作法:**最優解裡被翻的區間可假設兩兩不相交**(交換論證:取翻轉總長最小的最優解,兩重疊區間換成 `[min l,max l)`、`[min r,max r)` 朝向不變但總長更短)。於是化為把 1..N 切成 2K+1 段、奇數段取正面偶數段取反面、最大化總和 → `dp[i][j]` = 第 i 張屬第 j 段的前綴最大和,O(NK)。

- **[F. Many Mod Calculation](https://atcoder.jp/contests/abc466/tasks/abc466_f)** `數論` `dp` `~2300`
  題意:給 N,X 與正整數序列 A;對 x≥0 定義 `f(x)=(…((x mod A₁) mod A₂)…) mod A_N`;求 1..X 中 f(x)=0 的個數。T 筆,ΣN≤2×10⁵,X,Aᵢ≤10¹⁸。
  作法:官方——把值多重集記成若干「`[0,x)` 各 k 份」,套一次 `mod M` 把 `[0,x)` 拆成 ⌊x/M⌋ 份 `[0,M)` + 一份 `[0, x mod M)`;priority queue 合併同端點區間,因 **x mod M ≤ x/2** 每區間只生 O(log X) 段,總 O(N log N log X)。更好想的等價遞迴:先刪掉 Aᵢ≤Aᵢ₊₁(對後續 mod 無影響)使 A 嚴格遞減,令 g(X,t)=[0,X) 中通過第 t..N mod 鏈得 0 的個數,則 **g(X,t)=g(A_t,t+1)·⌊X/A_t⌋ + g(X mod A_t, t+1)**,後往前預算 g(Aᵢ,i+1);答案 = g(X+1,1)−1。

**不收**:~~G. Segment Sum Constraints~~(~3700)→ 超 ~2700 上限;二進位進位 DP(狀態 = 各前綴和條件的進位量)+ 勢能 DSU 判矛盾,組裝過於一次性,略過。
