# Codeforces Round 1104 (Order Capital Round 2, Div. 1 + Div. 2) — 學習重點

> 懶人包:每題「技巧 + 作法」。各難度都收主流可遷移的;偏門/太難略過。難度 `~` 為估計,據官方 editorial 整理。

- **[A. Destroying Towers](https://codeforces.com/contest/2237/problem/A)** `貪心` `~800`
  作法:維護**前綴最小值** `x = min(x, z)`,答案累加每一步的 `x`(`y += x`)。關鍵:每座塔貢獻的是「到目前為止的最小高度」。

- **[B. Annoying the Ghost](https://codeforces.com/contest/2237/problem/B)** `貪心` `逆序對` `~1200`
  作法:b 已排序;把每個 `a[i]` **貪心配到「不小於它的最小可用 `b[j]`」**(`set::lower_bound` 後刪除),無可配則 -1。所得序列的逆序對數即答案(n≤2000,O(n²) 即為正解)。貪心正確性:小的 b 留給大的 a 反而會卡死。

- **[C. Duck Surplus](https://codeforces.com/contest/2237/problem/C)** `貪心` `構造` `~1400`
  作法:由左到右,`if ans > a[i]: ans += a[i] else ans = a[i]`。**寶石:固定做最左可行操作後每個數命運唯一——它會一路向右「吞掉」比它小的後續元素**,等價於找以某數起算的最大可吞前綴和。

- **[D. Fullmetal Bitchemist](https://codeforces.com/contest/2237/problem/D)** `不變量` `組合計數` `~1700`
  作法:兩種操作都讓 (#1 − #0) 改變 ±3,故**`(#1 − #0) mod 3` 是不變量**——可縮成單字元 ⟺ 該差 ≢0 mod 3。用前綴和按 mod 3 配對計數美麗子串,再扣掉交替子串造成的重複計數(`-(y-1)//2`,`y` 為當前交替連續段長)。

- **[E. Permutation Commutation](https://codeforces.com/contest/2237/problem/E)** `排列` `置換環` `構造` `~2000`
  作法:**兩置換可交換 ⟺ B 把 A 的每個 cycle 映到同長度的 cycle**。先沿 A 的環傳播已知 b 值並檢查一致/重複,再把剩下 -1 的位置按「等長環」配對填入。

- **[F. Paint the Array](https://codeforces.com/contest/2237/problem/F)** `dp` `資料結構` `~2100`
  作法:序列合法 ⟺ `a_1=1, a_n=m`,且每相鄰對滿足 `a_{i+1}=a_i+1` 或 `a_i=m` 或 `a_{i+1}=1`。**`dp[i][j]` = 前綴以 j 結尾時最多保留幾個不改位置**,轉移用 shift + 區間/全域 remax(線段樹或前綴後綴 max)壓到 O(n+m);答案 `n − dp[n][m]`。

- **[G. Send GCDs](https://codeforces.com/contest/2237/problem/G)** `互動` `數論` `構造` `~2700`
  作法:用前 110 個質數的最大冪當「探針」,任意 b 元素對探針取 gcd 可還原其值,於是 2¹⁸ 個 smooth number 可被精確識別。**寶石:把 9 個 a 值(base 10⁶)轉成 10 個 smooth number(base k=2¹⁸)——`(10⁶)⁹ < k¹⁰`**,即進位制重編碼,陣列長壓到 ~10n/9。

- **[H. Slime and Queries](https://codeforces.com/contest/2237/problem/H)** `HLD` `ODT` `~3000`
  作法:樹剖把路徑拆成連續區段,用 `map`(ODT/區間賦值)維護每段顏色;路徑染色時逐段切割並統計被覆蓋數量,強制在線(`(p+ans)%n` 解碼下一詢問)。

**不收**:~~I1/I2. DBFS Order~~ → ~3300,樹形 dp + 容斥的硬計數,超過 ~2700 上限且非經典模板,短期投報低。

> 註:G ~2700 在上限上;**H ~3000 雖超標,但 HLD + 珂朵莉樹(ODT)是經典模板工具**,留作參考(經典模板可破例)。
