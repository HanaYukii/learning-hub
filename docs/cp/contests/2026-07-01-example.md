# 格式範例 — 學習重點(示意)

> digest 的格式示意;實際每場比賽各一檔。各難度都收主流可遷移的,偏門與太難略過。

- **E.(示意)帶修改的區間 DP 計數** `區間dp` `線段樹` `2500`
  - 洞察:轉移寫成矩陣後,區間答案 = 區間矩陣連乘;單點修改 → 線段樹維護矩陣乘積,`O(k³ log n)`/次。
  - 技巧:[線段樹維護矩陣乘積](/cp/techniques/segtree-matrix-product)。
  - 可遷移:任何「線性遞推 + 區間 + 點修改」都套這招。

- **F.(示意)樹上路徑異或第 k 大** `樹` `線性基` `2600`
  - 洞察:路徑 xor 用 dfs 前綴 xor;查詢轉成「把若干前綴值插進線性基後求最大」。
  - 易錯:合併線性基要消到 reduced 形,否則 max 查詢會錯。
