# AtCoder Regular Contest 224 — 學習重點

> 懶人包:每題「題意 + 核心作法」。各難度都收主流可遷移的;偏門/太難略過。難度 `~` 為 CF-equivalent。
>
> A 熱身略過($100K$ 必以 `00` 結尾,枚舉 $cK$、$1\le c\le100$ 取最小含 `00` 者)。從 B 開始列。

- **[B. Adjacent Tiles](https://atcoder.jp/contests/arc224/tasks/arc224_b)** `貢獻` `數學` `~1000`
  題意:平面上不重疊地放 $N$ 個等大正方形,求「恰好完整共用一整條邊」的方塊對數最大值($T$ 組,$N\le10^{18}$)。
  作法:**雙重計數周長**——所有方塊總周長 $4N$,其中恰「外圍邊界周長」被浪費,其餘每消耗 $2$ 單位周長換一對共邊,故答案 $=(4N-\text{邊界周長})/2$。最佳擺法是面積 $\ge N$ 中周長最小的近正方形包圍盒 $k\times k$ 或 $k\times(k+1)$(取 $k\times k<N$ 的最大 $k$),代入公式即得。

- **[C. Ascending Labels](https://atcoder.jp/contests/arc224/tasks/arc224_c)** `圖論` `構造` `~1400`
  題意:給連通簡單無向圖($N,M\le3\times10^5$),構造 $A$ 使 $A_1=0$、$0\le A_v\le N$,且每個 $v\ne1$ **恰有一個**鄰居 $w$ 滿足 $A_w=A_v-1$。
  作法:**取以頂點 $1$ 為根的 DFS 樹,$A_v=$ 該點深度即可**。非樹邊在 DFS 樹上必為返祖邊、兩端深度差 $\ge2$(差 $1$ 會是重邊),故 $v$ 的鄰居中深度 $=A_v-1$ 者只有父親一個,恰滿足條件,$O(N+M)$。

- **[D. Angst for All Pairs](https://atcoder.jp/contests/arc224/tasks/arc224_d)** `貪心` `組合計數` `~2000`
  題意:$N$ 張卡,每張可寫若干正整數,寫一次 $k$ 的成本 = $k$ 的十進位位數;要對每對 $1\le x<y\le K$ 都有某張卡恰含其一,問可行否、若可行求最小總成本($N,K\le10^6$)。
  作法:設 $S_x$=寫了 $x$ 的卡集合,條件 $\iff S_1,\dots,S_K$ 兩兩相異,故可行 $\iff K\le2^N$。成本 $=\sum_x|S_x|\cdot\mathrm{digits}(x)$,而尺寸 $|S_x|=k$ 至多可用 $\binom{N}{k}$ 次;**貪心把最小的可用尺寸配給位數最多(即最大)的 $x$**(交換論證),按位數段 × 二項式桶掃過即 $O(N+K)$。

- **[E. ABC|AB|A](https://atcoder.jp/contests/arc224/tasks/arc224_e)** `貪心` `字串` `~1800`
  題意:字串 $S$ 僅含 `A/B/C`,每次可刪掉一個為 `A`、`AB` 或 `ABC` 的連續子串,求可得的最短長度($T\le10^5$,$\sum|S|\le10^6$)。
  作法:**單調堆疊貪心**,由前往後掃:`A` 壓入(記狀態 $1$);遇 `B` 往下彈到見 $1$(A),有則併成 $2$(AB),無則此 `B` 永遠刪不掉、答案 $+1$;遇 `C` 彈到見 $2$(AB)一起刪去,無則答案 $+1$。刪不掉的字元數即最短長度,$O(|S|)$。
