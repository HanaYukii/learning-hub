# AtCoder Regular Contest 224 — 學習重點

> 懶人包:每題「題意 + 核心作法」。各難度都收主流可遷移的;偏門/太難略過。難度 `~` 為 CF-equivalent。
>
> A 熱身略過(求最小的 K 倍數、十進位含連續兩個 0:**100K 結尾必是 "00" 故答案 ≤ 100K**,枚舉 cK, c=1..100 取首個含 "00" 者)。從 B 起列。

- **[B. Adjacent Tiles](https://atcoder.jp/contests/arc224/tasks/arc224_b)** `數學` `構造` `~1000`
  題意:平面上不重疊放 N 個同大小正方形瓦片(N≤10¹⁸),求「完整共享一整條邊」的相異瓦片對數最大值。T 筆。
  作法:所有瓦片周長總和 4N;擺成一個連通塊時,**被「浪費」掉的周長恰等於包圍盒周長**,其餘每消耗 2 單位周長換一組共享邊 → 答案 = (4N − 包圍盒周長)/2 = 2N − 周長/2。取面積 ≥N 且周長最小的包圍盒:`k×k`、`k×(k+1)` 或 `(k+1)²`,k≈⌊√N⌋。O(1)。

- **[C. Ascending Labels](https://atcoder.jp/contests/arc224/tasks/arc224_c)** `構造` `圖論` `樹` `~1400`
  題意:給連通簡單無向圖(N 點 M 邊);構造 A:每點 0≤A_v≤N、A₁=0、且**每個 v≠1 恰有一個鄰居 w 滿足 A_w=A_v−1**。ΣN,ΣM≤3×10⁵。
  作法:以頂點 1 為根建 **DFS 樹**,令 A_v = DFS 深度即可。正當性:樹邊給每個非根點恰一個深度 −1 的父;**無向圖 DFS 樹沒有 cross edge,非樹邊必為祖先–後代且非直接父子(否則重邊),深度差 ≥2**,不會多造出一個 A_w=A_v−1 的鄰居。O(N+M)。

- **[D. Angst for All Pairs](https://atcoder.jp/contests/arc224/tasks/arc224_d)** `貪心` `組合計數` `~2000`
  題意:N 張卡,每卡可寫任意多個正整數,寫一次 k 成本 = k 的位數;要對所有 1≤x<y≤K 都存在某卡「恰含 x、y 之一」,求最小總成本,不可能輸出 −1。ΣN,ΣK≤10⁶。
  作法:令 S_x = 寫了 x 的卡集合,條件 ⟺ **S₁..S_K 兩兩相異**(某對相等則該對永遠分不開)→ 可行 ⟺ K≤2^N。成本 = Σ|Sᵢ|·digits(i),而大小為 k 的集合最多用 C(N,k) 次。**貪心:位數愈大的 i 配愈小的 |Sᵢ|**(交換論證),把「各位數的 i 個數」與「各 size 的 C(N,k) 容量」用雙指針對配累加。

- **[E. ABC|AB|A](https://atcoder.jp/contests/arc224/tasks/arc224_e)** `貪心` `字串` `~1800`
  題意:給只含 A/B/C 的字串 S;可任意次刪除等於 `A`、`AB` 或 `ABC` 的連續子串(刪後接合);求最終最短長度。Σ|S|≤10⁶。
  作法:前向掃描 + **stack** 貪心:A→壓「1」(狀態 A);B→往下彈到出現 1 則改壓「2」(AB),否則此 B 永遠刪不掉、答案 +1;C→往下彈到出現 2 則整組 ABC 一起消,否則此 C 刪不掉、答案 +1。**可組成 A/AB/ABC 的字元隨時能一起清掉;湊不成的那個字元把序列切成前後獨立兩段**。答案 = 不可刪字元數,O(|S|)。

**不收**:~~F. AND/OR~~(~2800)→ 超 ~2700 上限;逆序維護「確定旗標 f」證明每步只剩至多 2 種後綴,再用帕斯卡三角前綴和 O(1) 遞推計數,思路一次性,略過。
