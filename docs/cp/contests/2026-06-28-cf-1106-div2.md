# Codeforces Round 1106 (Div. 2) — 學習重點

> 懶人包:每題「題意 + 核心作法」。難度 `~` 為 CF-equivalent。

A(Another Puzzle from Papyrus,~800):題意:給定兩個長度 n 的整數陣列 a、b 與整數 c(n、c、值皆 ≤ 100)。每次操作可任選 i 把 aᵢ 減 1(花 1 秒),或將 a 任意重排(花 c 秒)。求/輸出:把 a 變成 b 的最少總秒數;不可能則 −1。作法:減量總和固定 = **Σ(aᵢ−bᵢ)**;原序若有 aᵢ<bᵢ 則加一次 c、兩排各自排序後重驗,仍有 aᵢ<bᵢ 輸出 −1。純實作,熱身略過。

- **[B. Crimson Triples](https://codeforces.com/contest/2238/problem/B)** `數論` `貢獻` `~1100`
  題意:給定整數 n(n≤2×10⁵)。有序三元組 (a,b,c) 滿足 1≤a,b,c≤n(可重複),若 gcd(lcm(a,b), lcm(b,c)) = gcd(a,c) 則稱為 crimson。
  求/輸出:crimson 三元組的總數;兩三元組只要任一位置的值不同即視為不同。
  作法:條件**等價於 b 同時整除 a 與 c**(兩個 lcm 都是 b 的倍數 ⇒ b | 左式 = gcd(a,c);反向代入顯然);固定 b 後 a、c 各 ⌊n/b⌋ 種,答案 = **Σ_{b=1}^{n} ⌊n/b⌋²**,O(n)(整除分塊可 O(√n))。

- **[C. Village Guilds](https://codeforces.com/contest/2238/problem/C)** `樹` `dp` `~1400`
  題意:給定一棵 n 個點、以 1 為根的有根樹(n≤2×10⁵)。對每個點 v 與非負整數 h,「公會」定義為 v 的子樹中與 v 距離恰為 h 的點所成的集合(h=0 時即 {v}),兩公會不同若且唯若集合不同。
  求/輸出:整棵樹中相異非空公會的個數。
  作法:每點**只需子分支 top-2 深度**:h 超過次深分支的層完全落在最深分支內、已在該子樹計過;其餘層都是新公會,累加「次深 − depth(v) + 1」,一次 DFS O(n)。(等價:逐層把同層點按 DFS 序排,數相鄰點的相異 LCA。)

- **[D. Storming Arasaka](https://codeforces.com/contest/2238/problem/D)** `數論` `構造` `~1700`
  題意:每筆詢問給定整數 n(n≤10⁶,t≤10⁴)。把 n 的所有大於 1 的因數(含 n 本身)分成若干非空有序層 L₁,…,L_k,須滿足:任一數 x 的因數(1 與 x 自身除外)全落在比 x 所在層更早的層,且每層內所有數能排成一條相鄰兩數 gcd>1 的鏈。
  求/輸出:滿足條件的最少層數 k。
  作法:答案 = **Ω(n) + ω(n) − 1**。下界:因數鏈 [p₁],[p₁²],…,[n] 逼出 Ω 層,且**含質數的層必為單元素層**(鏈上鄰居必為其倍數、須排更晚層),ω 個質數再多貢獻 ω−1 層;按 Ω 值分層可構造達成。線性篩後 O(1) 查詢。(n=120=2³·3·5 → 5+3−1=7。)

- **[E. Cake Trial](https://codeforces.com/contest/2238/problem/E)** `dp` `狀態擴充` `~2000`
  題意:給定長度 n 的字串 s,字元為 T(真)/F(假)/N(未定)(n≤500,Σn³≤500³)。對手先把每個 N 填成 T 或 F;你看到最終字串後,選一段連續區間(可空)宣稱「區間內全 F、區間外全 T」,錯誤數 = 區間外的 F 數 + 區間內的 T 數,你會選錯誤數最少的區間。
  求/輸出:對手以最大化你的最少錯誤數為目標填 N,求該最大值。
  作法:固定盤面後最少錯誤 = **#F − 最大子段和**(F=+1、T=−1、可空)。**把 clamp-at-0 的 Kadane 現值當 DP 維度**:狀態 (i, cnt_F, cur) 存最小可達的歷史最大子段和,答案取 max(cnt_F − dp),O(n³);另解:枚舉子段和上界 D、由左往右貪心壓 T 數,O(n²)。

- **[F. Infinite Work](https://codeforces.com/contest/2238/problem/F)** `組合計數` `反射原理` `~2300`
  題意:近乎無限的完全二元樹(1 為根,點 i 的兒子為 2i、2i+1),初始所有點皆在職。每天兩階段:先「補雇」——每個在職點讓其不在職的直接兒子復職(當天新復職者不再雇人);再「開除」——任選一組在職點直接開除,被除者整棵子樹連鎖被開除(1 不可除,且每點終生至多被開除一次)。給定 n, k(n≤10⁹,k≤2×10⁵,t≤10⁵)。
  求/輸出:n 天後恰剩 k 個在職點、且直接開除總次數最少的開除方案數 mod 1e9+7。
  作法:令 K=k+1,把 K 的二進位中位 ≥n 的位反覆「拆成兩個低一位」壓到位 <n(允許重複),得 sm 個 2 的冪、位 i 出現 bits[i] 次。答案 = **Catalan(sm−1) · sm!/∏ bits[i]!**——即以這組 2 的冪為葉子的相異二元樹數(Catalan 由反射原理 C(2m,m)−C(2m,m−1) 求)。預處理階乘/逆階乘,每詢問 O(log)。
