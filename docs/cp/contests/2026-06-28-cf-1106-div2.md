# Codeforces Round 1106 (Div. 2) — 學習重點

> 懶人包:每題「技巧 + 作法」。各難度都收主流可遷移的;偏門/太難略過。難度 `~` 為 CF-equivalent。

A(Another Puzzle from Papyrus,~800):答案與順序無關 = **Σ(aᵢ−bᵢ)**;原序若有 aᵢ<bᵢ 則加一次固定成本 C 並排序兩排重驗,仍匹配不上輸出 −1。純實作,熱身略過。

- **[B. Crimson Triples](https://codeforces.com/contest/2238/problem/B)** `數論` `貢獻` `~1100`
  作法:數三元組 (a,b,c)(1≤a,b,c≤n)滿足 gcd(lcm(a,b),lcm(b,c))=gcd(a,c)。關鍵:此式**等價於 b 同時整除 a 與 c**;固定 b 後 a、c 各有 ⌊n/b⌋ 種,答案 = **Σ_{b=1}^{n} ⌊n/b⌋²**,O(n)(或整除分塊 O(√n))。

- **[C. Village Guilds](https://codeforces.com/contest/2238/problem/C)** `樹` `dp` `~1400`
  作法:有根樹上求最多公會數。一次 DFS 回傳子樹最大深度,每個節點**只看最深與次深兩條子分支**:最深那條免費往上併,其餘分岔各自成新公會 → 累加「次大子樹深度 − 該點深度 + 1」。**只需維護 top-2 深度**,O(n)。(等價寫法:按 DFS 序把相鄰節點的相異 LCA 累加。)

- **[D. Storming Arasaka](https://codeforces.com/contest/2238/problem/D)** `數論` `構造` `~1700`
  作法:把 n 的因數依「同組不互為倍數 + 連通」規則分層,求最少層數。結論:答案 = **Ω(n) + ω(n) − 1**(帶重數的質因子總數 + 相異質數個數 − 1)。理由:沿因數鏈 [p₁],[p₁²],…,[p₁^{a₁}],[p₁^{a₁}p₂],…,[n] 長 Ω(n) 強制至少這麼多層;ω 個相異質數又須各自獨立起層(已含 p₁,故 +(ω−1))。線性篩最小質因子後 O(1) 查詢。(n=120=2³·3·5 → 5+3−1=7。)

- **[E. Cake Trial](https://codeforces.com/contest/2238/problem/E)** `dp` `狀態擴充` `~2000`
  作法:字串每個 '?' 指定成 F(+1)或 T(−1)(固定字元不可改),最大化「**F 個數 − 最大子段和**」。DP 狀態 (位置, 已放 F 數 cnt_f, 目前 clamp-Kadane 值 cur_s),值存「已達成的最小最大子段和」:放 F 時 cur_s+1 並用 max 更新代價,放 T 時 cur_s−1 且 **clamp 到 0**、代價不變;答案取 max(cnt_f − dp)。**把 clamp-at-0 的 Kadane 前綴當成 DP 維度**,O(n²)。另解:枚舉上界 D、貪心最小化 T 數,同為 O(n²)。

- **[F. Infinite Work](https://codeforces.com/contest/2238/problem/F)** `組合計數` `反射原理` `~2300`
  作法:令 K=k+1,把 K 拆成一堆 <2ⁿ 的 2 的次方(高位不斷「除 2 拆兩份」壓到位 < n,允許重複),設共 sm 個、位 i 出現 bits[i] 次。答案 = **Catalan(sm−1) · 多項式係數 sm!/∏ bits[i]!** —— 即以這組 2 的次方為葉子的相異二元樹數量(Catalan 由反射原理 C(2m,m)−C(2m,m−1) 求)。預處理階乘/逆階乘,每詢問 O(log)。
