# Codeforces Round 1105 (Div. 1) — 學習重點

> 懶人包:每題只記「**技巧 + 作法**」。各難度都收主流可遷移的;偏門/太難略過(見底)。難度 `~` 為估計。
> (據官方 Hints + 參考 code 重建;散文題解在 JS spoiler 裡抓不到。判讀有誤回報我修。)

- **[A. Nim Game Is XOR Game](https://codeforces.com/contest/2239/problem/A)** `博弈` `XOR` `~1300`
  作法:全體 XOR $=s$;答案 = 含 $s$ 最高位的元素數(致勝首步數)。`s^a_i < a_i ⇔ a_i 含 s 的最高 set bit`。

- **[B. Decidophobia](https://codeforces.com/contest/2239/problem/B)** `貪心` `貢獻` `~1800`
  作法:貢獻重構——每位置淨貢獻 $(2d{+}1)a_i-(\text{半徑 }d\text{ 窗和})$,獨立取正相加;環狀 → 複製三倍 + 前綴和。

- **[C. Revival](https://codeforces.com/contest/2239/problem/C)** `構造` `BIT` `~2300`
  作法:從後往前定值,「把前面比它大的各 −1」使 $p$ 同步減、$s$ 不變 → 塌縮成定最後一個;BIT 求第 $k$ 小重建排列。

- **[D. Hunting the Beast](https://codeforces.com/contest/2239/problem/D)** `組合計數` `容斥` `數論` `~2700`
  作法:對稱性 → 算固定集合成功的圖數 $\times\binom{n}{m}$,容斥拆。**寶石恆等式**:$n$ 點拆不相交無自環環、按環數帶 $(-1)^k$,總和 $=1-n$。線性篩算 $i^{\,n-1}$。

- **[E. The end of this world](https://codeforces.com/contest/2239/problem/E)** `離線` `線段樹分治` `可回滾DSU` `~2900`
  作法:邊存在於時間區間 $[\mathrm{low}_i,w_i]$ → **時間軸線段樹分治 + 可回滾並查集**,維護連通塊內 max。動態圖連通性離線框架。

**不收**:~~F. Colorful Works~~ → mod 2 DP→賽局轉化 + 容斥 + bitset + 多項式 $(1+x^k)^2{=}1{+}x^{2k}$,太硬又偏門,短期用不上。
