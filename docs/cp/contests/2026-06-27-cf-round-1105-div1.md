# Codeforces Round 1105 (Div. 1) — 學習重點

> 懶人包:每題只記「**題意 + 核心作法**」。各難度都收主流可遷移的;偏門/太難略過(見底)。難度 `~` 為估計。
> (作法據官方 Hints + 參考 code 重建,題意據原題面;散文題解在 JS spoiler 裡抓不到。判讀有誤回報我修。)

- **[A. Nim Game Is XOR Game](https://codeforces.com/contest/2239/problem/A)** `博弈` `XOR` `~1300`
  題意:給長 $n$ 的非負陣列($n\le 10^6$,$a_i<2^{30}$),每回合選不全零的 $b$($0\le b_i\le a_i$ 且 $\oplus\, b_i=0$)並令 $a_i\mathrel{-}=b_i$,無法操作者輸;求 Alice 必勝首步 $b$ 的數量 mod 998244353。
  作法:必敗態 ⇔ 至多一個非零元素,必勝首步 = 一步打進必敗態:留 $a_j$ 獨活需 $b_j=s\oplus a_j<a_j$($s$ 為全體 XOR)⇔ **$a_j$ 含 $s$ 的最高 set bit**;$s=0$ 時唯一贏法是整盤拿光(答案 1);$n=1$ 無合法步,答案 0。

- **[B. Decidophobia](https://codeforces.com/contest/2239/problem/B)** `貪心` `貢獻` `~1800`
  題意:$n$ 人圓桌($n\le 2\cdot 10^5$),視野為順逆時針各 $d$ 人($d<\frac n2$);發禮物給任意子集,拿到者得(視野內沒拿人數)$\cdot a_i$,沒拿者失(視野內有拿人數)$\cdot a_i$,最大化總快樂。
  作法:貢獻重排——選 $i$ 的淨貢獻恆為 $(2d{+}1)a_i-(\text{半徑 }d\text{ 含自身窗和})$,**與其他人選誰完全無關** → 各位置獨立,取正者相加。

- **[C. Revival](https://codeforces.com/contest/2239/problem/C)** `構造` `BIT` `~2300`
  題意:未知排列 $p$($n\le 2\cdot 10^5$),$s_i$ 為前綴 $i$ 的逆序數;每個位置 $i$ 恰給一條 $p_i=x$ 或 $s_i=x$ 的限制,輸出任一合法排列(保證存在)。
  作法:從後往前只定「目前最後一位」——**定下它後把前面比它大的值全 $-1$:前綴仍是排列、每個 $s_i$ 都不變** → 問題塌縮成規模 $-1$;可用值集合用 BIT 維護(查名次/第 $k$ 小)重建。

- **[D. Hunting the Beast](https://codeforces.com/contest/2239/problem/D)** `組合計數` `容斥` `數論` `~2700`
  題意:$n$ 點 functional graph(每點恰一條出邊、禁自環,共 $(n-1)^n$ 種;$n\le 10^6$);大小 $m$ 的起點集 $S$ 成功 ⇔ 每個點都可自 $S$ 中某點到達;求所有圖的成功 $S$ 數總和 mod 998244353。
  作法:對稱性 → 固定一組 $S$ 算成功圖數 $\times\binom{n}{m}$,成功條件容斥拆。**寶石恆等式**:$n$ 點拆成不相交無自環環、$k$ 個環計 $(-1)^k$,總和恆為 $1-n$;二項式定理化簡,線性篩批量算 $i^{\,n-1}$。

- **[E. The end of this world](https://codeforces.com/contest/2239/problem/E)** `離線` `線段樹分治` `可回滾DSU` `~2900`
  題意:無向圖($n,m\le 5\cdot 10^5$),點有值 $\mathrm{val}$,邊有容量 $w$ 與下限 $\mathrm{low}$;自選初始 $h$,$h\le w_j$ 才能走邊 $j$,走後 $h\leftarrow\max(h,\mathrm{low}_j)$;走 $\ge 1$ 條邊停於 $t$ 得 $\mathrm{val}_t+h_{start}$,對每個起點求最大得分(走不動輸出 $-1$)。
  作法:**把 $h$ 當時間軸:邊 $j$ 恰於時刻區間 $[\mathrm{low}_j,w_j]$ 存活** → 離線動態連通性:時間軸線段樹分治 + 可回滾並查集,由大到小時刻轉移,維護連通塊內 max 並支援整塊 chmax。
