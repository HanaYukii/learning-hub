### [A. Nim Game Is XOR Game](https://codeforces.com/contest/2239/problem/A)
`博弈` `XOR` `~1300`

::: info 題意
給定長 $n$ 的非負整數陣列 $a$($n\le 10^6$,$a_i<2^{30}$),Alice 與 Bob 輪流操作,Alice 先手。

每回合當前玩家選一個長 $n$ 的非負整數陣列 $b$,須滿足 $0\le b_i\le a_i$、$b$ 不全為零、且 $b_1\oplus b_2\oplus\cdots\oplus b_n=0$,然後令 $a_i\leftarrow a_i-b_i$;選不出合法 $b$ 的玩家輸。

求:雙方最優下,Alice 第一步能保證必勝的 $b$ 有幾種,輸出個數 mod $998244353$。

例:`a = [2, 3]` → Alice 選 `b = [2, 2]`($2\oplus 2=0$ 合法)後剩 `[0, 1]`:Bob 的 $b$ 只有 $b_2$ 能非零,XOR 無法歸零 → Bob 無步可走。可驗證這是唯一必勝首步,答案 $1$。
:::

::: tip 作法
必敗態 ⇔ 至多一個非零元素,必勝首步 = 一步打進必敗態:留 $a_j$ 獨活需 $b_j=s\oplus a_j<a_j$($s$ 為全體 XOR)⇔ **$a_j$ 含 $s$ 的最高 set bit**。

$s=0$ 時唯一贏法是整盤拿光(答案 1);$n=1$ 無合法步,答案 0。

```cpp
// sketch:必勝首步計數
int s = 0; long long cnt = 0;
for (int x : a) s ^= x;
if      (n == 1) cnt = 0;            // 無合法步
else if (s == 0) cnt = 1;            // 唯一贏法:整盤拿光
else {
    int hb = 31 - __builtin_clz(s);  // s 的最高 set bit
    for (int x : a)
        if (x >> hb & 1) cnt++;      // 留 a_j 獨活:b_j = s ^ a_j < a_j
}
```
:::

### [C. Revival](https://codeforces.com/contest/2239/problem/C)
`構造` `BIT` `~2300`

::: info 題意
有一個未知的 $1\sim n$ 排列 $p$($n\le 2\cdot 10^5$),令 $s_i$ 為前綴 $p_1,\ldots,p_i$ 的逆序對數。

輸入對每個位置 $i$ 恰好給一條限制,形如 $p_i=x$ 或 $s_i=x$。輸出:任一同時滿足全部 $n$ 條限制的排列(保證有解)。

例:排列 `p = [3, 1, 2]` 的前綴逆序對為 `s = [0, 1, 2]`;若三條限制是 `p_1 = 3`、`s_2 = 1`、`s_3 = 2`,它就是一組合法輸出。
:::

::: tip 作法
從後往前只定「目前最後一位」——**定下它後把前面比它大的值全 $-1$:前綴仍是排列、每個 $s_i$ 都不變** → 問題塌縮成規模 $-1$。

可用值集合用 BIT 維護(查名次/第 $k$ 小)重建。

```cpp
// sketch:主迴圈 —— BIT 維護剩餘可用值(rank / kth)
for (int i = n; i >= 1; --i) {   // 從後往前,每輪定出目前最後一位
    int r;                       // p_i 在剩餘值中的名次
    if (con[i].is_p) r = bit.rank(con[i].x);  // p_i = x:查 x 的目前名次
    else             r = rank_from_s(i);      // s_i = x:由逆序對數反推
    p[i] = bit.kth(r);           // 名次 → 第 r 小的剩餘值
    bit.erase(p[i]);             // 刪除後大值名次自動 -1 ⇔ 題解的「全 -1」
}
```
:::

### [D. Hunting the Beast](https://codeforces.com/contest/2239/problem/D)
`組合計數` `容斥` `數論` `~2700`

::: info 題意
給定整數 $n,m$($n\le 10^6$)。考慮 $n$ 點 functional graph:每點恰有一條出邊、禁止自環,出邊終點於其餘 $n-1$ 點中任選,共 $(n-1)^n$ 種圖。

對一張圖,大小 $m$ 的起點集 $S$ 稱為成功 ⇔ 圖中每個點都能從 $S$ 內某點沿有向邊到達(點視為可達自身);一張圖的價值 = 其成功 $S$ 的個數。

求:全部 $(n-1)^n$ 張圖的價值總和,輸出 mod $998244353$。

例:`n=3, m=1` → 共 $2^3=8$ 張圖;記 $f(v)$ 為 $v$ 的出邊終點,則 $S=\{v\}$ 成功 ⇔ $v\to f(v)\to f(f(v))$ 恰好走遍三點(即 $f(f(v))\neq v$)。兩張三元環圖各有 $3$ 個成功起點,其餘 $6$ 張各 $1$ 個 → 總和 $2\cdot 3+6\cdot 1=12$。
:::

::: tip 作法
對稱性 → 固定一組 $S$ 算成功圖數 $\times\binom{n}{m}$,成功條件容斥拆。

**寶石恆等式**:$n$ 點拆成不相交無自環環、$k$ 個環計 $(-1)^k$,總和恆為 $1-n$;二項式定理化簡,線性篩批量算 $i^{\,n-1}$。
:::

### [E. The end of this world](https://codeforces.com/contest/2239/problem/E)
`離線` `線段樹分治` `可回滾DSU` `~2900`

::: info 題意
給定 $n$ 點 $m$ 邊無向圖($n,m\le 5\cdot 10^5$),點 $i$ 有值 $\mathrm{val}_i$,邊 $j$ 有容量 $w_j$ 與下限 $\mathrm{low}_j$($w_j\ge\mathrm{low}_j$)。

出發前自選任一非負整數 $h_{start}$ 作初始狀態;狀態為 $h$ 時可走邊 $j$ ⇔ $w_j\ge h$,走過後 $h\leftarrow\max(h,\mathrm{low}_j)$。

走至少一條邊後停於點 $t$,得分為 $\mathrm{val}_t+h_{start}$(加的是初始值,非最終 $h$)。輸出:對每個起點 $s=1,\ldots,n$ 的最大得分;若從 $s$ 一條邊都走不了則輸出 $-1$。

例:3 點,邊 `1–2 (w=5, low=3)` 與 `2–3 (w=2, low=0)`,`val = [0, 0, 100]` → 從 $1$ 出發只有 1–2 可走(需 $h_{start}\le 5$),走完後 $h\ge 3>2=w_{2\text{–}3}$ → 永遠到不了 $3$,$100$ 拿不到;最佳取 $h_{start}=5$ 停在 $2$,得 $0+5=5$。
:::

::: tip 作法
**把 $h$ 當時間軸:邊 $j$ 恰於時刻區間 $[\mathrm{low}_j,w_j]$ 存活** → 離線動態連通性。

時間軸線段樹分治 + 可回滾並查集,由大到小時刻轉移,維護連通塊內 max 並支援整塊 chmax。

```cpp
// sketch:線段樹分治 DFS 骨架
void dfs(int u, int l, int r) {
    size_t snap = dsu.history();                      // 記錄回滾點
    for (auto [x, y] : edges_at[u]) dsu.unite(x, y);  // 掛在此節點的邊
    if (l == r) collect(l);                           // 時刻 l:收割/塊內轉移
    else { int m = (l + r) / 2;
           dfs(rson, m + 1, r); dfs(lson, l, m); }    // 由大到小時刻轉移
    dsu.rollback(snap);                               // 回滾
}
```
:::
