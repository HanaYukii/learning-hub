# Codeforces Round 1105 (Div. 1) — 學習重點

- **[A. Nim Game Is XOR Game](https://codeforces.com/contest/2239/problem/A)** `博弈` `XOR` `~1300`
  題意:給定長 $n$ 的非負整數陣列 $a$($n\le 10^6$,$a_i<2^{30}$),Alice 與 Bob 輪流操作,Alice 先手。每回合當前玩家選一個長 $n$ 的非負整數陣列 $b$,須滿足 $0\le b_i\le a_i$、$b$ 不全為零、且 $b_1\oplus b_2\oplus\cdots\oplus b_n=0$,然後令 $a_i\leftarrow a_i-b_i$;選不出合法 $b$ 的玩家輸。求:雙方最優下,Alice 第一步能保證必勝的 $b$ 有幾種,輸出個數 mod $998244353$。
  作法:必敗態 ⇔ 至多一個非零元素,必勝首步 = 一步打進必敗態:留 $a_j$ 獨活需 $b_j=s\oplus a_j<a_j$($s$ 為全體 XOR)⇔ **$a_j$ 含 $s$ 的最高 set bit**;$s=0$ 時唯一贏法是整盤拿光(答案 1);$n=1$ 無合法步,答案 0。

- **[B. Decidophobia](https://codeforces.com/contest/2239/problem/B)** `貪心` `貢獻` `~1800`
  題意:給定圓桌旁順時針編號的 $n$ 個人($n\le 2\cdot 10^5$),每人有權重 $a_i$,以及公共視野參數 $d$($d<\frac n2$):$i$ 的視野為順、逆時針各 $d$ 人(共 $2d$ 人,不含自己)。你可發禮物給任意子集:拿到者若視野內有 $x$ 人沒拿到,得 $x\cdot a_i$ 快樂;沒拿到者若視野內有 $x$ 人拿到,失 $x\cdot a_i$ 快樂。輸出:全體快樂總和的最大值。
  作法:貢獻重排——選 $i$ 的淨貢獻恆為 $(2d{+}1)a_i-(\text{半徑 }d\text{ 含自身窗和})$,**與其他人選誰完全無關** → 各位置獨立,取正者相加。

- **[C. Revival](https://codeforces.com/contest/2239/problem/C)** `構造` `BIT` `~2300`
  題意:有一個未知的 $1\sim n$ 排列 $p$($n\le 2\cdot 10^5$),令 $s_i$ 為前綴 $p_1,\ldots,p_i$ 的逆序對數。輸入對每個位置 $i$ 恰好給一條限制,形如 $p_i=x$ 或 $s_i=x$。輸出:任一同時滿足全部 $n$ 條限制的排列(保證有解)。
  作法:從後往前只定「目前最後一位」——**定下它後把前面比它大的值全 $-1$:前綴仍是排列、每個 $s_i$ 都不變** → 問題塌縮成規模 $-1$;可用值集合用 BIT 維護(查名次/第 $k$ 小)重建。

- **[D. Hunting the Beast](https://codeforces.com/contest/2239/problem/D)** `組合計數` `容斥` `數論` `~2700`
  題意:給定整數 $n,m$($n\le 10^6$)。考慮 $n$ 點 functional graph:每點恰有一條出邊、禁止自環,出邊終點於其餘 $n-1$ 點中任選,共 $(n-1)^n$ 種圖。對一張圖,大小 $m$ 的起點集 $S$ 稱為成功 ⇔ 圖中每個點都能從 $S$ 內某點沿有向邊到達(點視為可達自身);一張圖的價值 = 其成功 $S$ 的個數。求:全部 $(n-1)^n$ 張圖的價值總和,輸出 mod $998244353$。
  作法:對稱性 → 固定一組 $S$ 算成功圖數 $\times\binom{n}{m}$,成功條件容斥拆。**寶石恆等式**:$n$ 點拆成不相交無自環環、$k$ 個環計 $(-1)^k$,總和恆為 $1-n$;二項式定理化簡,線性篩批量算 $i^{\,n-1}$。

- **[E. The end of this world](https://codeforces.com/contest/2239/problem/E)** `離線` `線段樹分治` `可回滾DSU` `~2900`
  題意:給定 $n$ 點 $m$ 邊無向圖($n,m\le 5\cdot 10^5$),點 $i$ 有值 $\mathrm{val}_i$,邊 $j$ 有容量 $w_j$ 與下限 $\mathrm{low}_j$($w_j\ge\mathrm{low}_j$)。出發前自選任一非負整數 $h_{start}$ 作初始狀態;狀態為 $h$ 時可走邊 $j$ ⇔ $w_j\ge h$,走過後 $h\leftarrow\max(h,\mathrm{low}_j)$。走至少一條邊後停於點 $t$,得分為 $\mathrm{val}_t+h_{start}$(加的是初始值,非最終 $h$)。輸出:對每個起點 $s=1,\ldots,n$ 的最大得分;若從 $s$ 一條邊都走不了則輸出 $-1$。
  作法:**把 $h$ 當時間軸:邊 $j$ 恰於時刻區間 $[\mathrm{low}_j,w_j]$ 存活** → 離線動態連通性:時間軸線段樹分治 + 可回滾並查集,由大到小時刻轉移,維護連通塊內 max 並支援整塊 chmax。
