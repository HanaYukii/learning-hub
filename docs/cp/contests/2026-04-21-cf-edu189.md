# Educational Codeforces Round 189 — 學習重點

> 難度 `~` 為 CF 官方 rating。

### [A. A Number Between Two Others](https://codeforces.com/contest/2225/problem/A)
`數學` `~800`

::: info 題意
給定整數 $x,y$,保證 $y>x$ 且 $x\mid y$。

**求**:是否存在 $z$ 滿足 $x<z<y$、$x\mid z$、$y\bmod z\neq 0$;輸出 YES/NO。

例:$x=2,y=4$ → $(2,4)$ 內無 $2$ 的倍數 → NO;$x=2,y=6$ → $z=4$:$6\bmod 4=2\neq 0$ → YES。
:::

::: tip 作法
**YES ⇔ $y>2x$**:記 $q=y/x\ge 3$,$q$ 奇取 $z=2x$($q/2$ 非整數 ⇒ $2x\nmid y$),$q$ 偶取 $z=(q-1)x$(餘恰為 $x$);$y=2x$ 時區間內根本沒有 $x$ 的倍數。
:::

### [B. Alternating String](https://codeforces.com/contest/2225/problem/B)
`不變量` `字串` `~900`

::: info 題意
給定僅含 a/b 的字串 $s$。**至多一次**操作:選一段子字串,可選擇是否整段反相(a↔b),然後**必須**整段反轉。

**求**:能否使 $s$ 變成相鄰字元皆不同的交替字串;輸出 YES/NO。

例:`aabb` → 選中段 `ab` 反轉成 `ba`,得 `abab` ✓ → YES。
:::

::: tip 作法
**反轉+反相都不改變子串內部「相鄰相等」關係,只可能修掉所選兩端點處的壞對** → 一次操作至多修 2 個壞對,且 ≤2 個時必可構造:YES ⇔ 相鄰相等對數 ≤ 2。
:::

### [C. Red-Black Pairs](https://codeforces.com/contest/2225/problem/C)
`dp` `~1100`

::: info 題意
給定 $2\times n$ 表格,每格紅或黑。你要重繪若干格,使全部格子能分成 $n$ 對,每對**同色且共邊**(即同色骨牌密鋪)。

**求**:最少重繪格數。

例:上排 `RR`、下排 `BR` → 第 1 欄直配 $(R,B)$ 重繪 1 格、第 2 欄直配 $(R,R)$ 免費 → 答案 $1$。
:::

::: tip 作法
密鋪結構只有兩種前進方式:**這欄放 1 張直骨牌(代價 $[a_i\neq b_i]$),或連兩欄放 2 張橫骨牌(代價 $[a_i\neq a_{i+1}]+[b_i\neq b_{i+1}]$)**——一維 dp 直接掃。

```cpp
// sketch:dp[i] = 前 i 欄配好對的最少重繪
dp[0] = 0;
for (int i = 0; i < n; i++) {
    dp[i+1] = min(dp[i+1], dp[i] + (a[i] != b[i]));          // 直的一對
    if (i + 1 < n)                                            // 兩張橫的
        dp[i+2] = min(dp[i+2], dp[i] + (a[i] != a[i+1]) + (b[i] != b[i+1]));
}
// 答案 dp[n]
```
:::

### [D. Exceptional Segments](https://codeforces.com/contest/2225/problem/D)
`位元` `XOR` `數學` `~1500`

::: info 題意
給定 $n,x$。考慮序列 $1,2,\dots,n$。

**求**:滿足 $1\le l\le x\le r\le n$ 且 $l\oplus(l{+}1)\oplus\cdots\oplus r=0$ 的 $(l,r)$ 對數,輸出 mod $998\,244\,353$。

例:$n=7,x=6$ → 恰有 $(4,7)$ 與 $(1,7)$($4\oplus 5\oplus 6\oplus 7=0$、$1\oplus\cdots\oplus 7=0$)→ 答案 $2$。
:::

::: tip 作法
**前綴 XOR $f(m)=1\oplus\cdots\oplus m$ 按 $m\bmod 4$ 循環取 $m,\,1,\,m{+}1,\,0$**($m\equiv 0,1,2,3$);區間 XOR 為 0 ⇔ $f(l{-}1)=f(r)$,而非常數兩支($m$、$m{+}1$)值不重複 → **等值只能同落在 $f=0$ 類($\{0\}\cup\{m\equiv 3\}$)或 $f=1$ 類($m\equiv 1$)**。

跨 $x$ 兩側獨立:$\text{ans}=c_0^{[0,x-1]}c_0^{[x,n]}+c_1^{[0,x-1]}c_1^{[x,n]}$,計數一行公式 $\#\{m<R: m\equiv j\ (\text{mod }4)\}=\lfloor R/4\rfloor+[R\bmod 4>j]$,$O(1)$。
:::

```cpp
// sketch:隨機平移六角格,重抽直到覆蓋 >= 89%
int w = 2 * r, h = ceil(sqrt(3) * r);             // 列寬、行距
do {
    long long x0 = rng() % 100000, y0 = rng() % 100000;  // 隨機平移
    cov = 0; ans.clear();

        if (nearby_circle_covers(x, y)) { record_center(); cov++; }
} while (cov * 100 < 89 * n);                      // 期望 ~90.7%,少數輪即過
```

### [F. String Cutting](https://codeforces.com/contest/2225/problem/F)
`字串` `貪心` `二分` `~2600`

::: info 題意
給定小寫字串 $s$(長 $n$)與 $l,k$。把 $s$ 切成**至少 $k$ 段**、每段長度至少 $l$;然後把所有段按字典序排序。

**求**:讓排序後**第 $k$ 名**的字串字典序最大;輸出該字串($l\cdot k>n$ 時無解輸出 NO)。

例:$s=$ `abracadabra`、$l=2,k=5$ → 切 `ab|rac|ad|ab|ra`,排序後為 `ab,ab,ad,ra,rac`,第 5 名 `rac`,可證最優。
:::

::: tip 作法
**候選結構:第 $k$ 名可設為 $s[i..j]$——前綴 $[0,i)$ 切滿 $\lfloor i/l\rfloor$ 塊,尾端只保留 $\max(0,\,k{-}1{-}\lfloor i/l\rfloor)$ 塊長度恰 $l$ 的墊底塊,候選吃掉中間全部**(同起點越長越大:短的是長的前綴)。枚舉起點得 $O(n)$ 個候選,取字典序最大——hash + 二分 LCP 每次比較 $O(\log n)$。$k=1$ 直接輸出整串。

```cpp
// sketch:枚舉第 k 名起點 i,固定候選結構後取字典序最大
int la = 0, ra = l - 1;                        // 目前最佳候選 [la, ra]
for (int i = 0; i < n; i++) {
    int cnt = i / l;                           // 前綴 [0,i) 最多切幾塊
    if (i > 0 && cnt == 0) continue;           // 前綴切不出一塊
    long long res = max(0LL, (long long)k - 1 - cnt) * l;  // 尾端墊底總長
    int j = n - res - 1;                       // 候選 [i, j]:越長越大
    if (j - i + 1 >= l && cmp(i, j, la, ra) > 0)   // hash + 二分 LCP
        { la = i; ra = j; }
}
```
:::
