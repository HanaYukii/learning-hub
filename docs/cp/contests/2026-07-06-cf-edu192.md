# Educational Codeforces Round 192 — 學習重點

> 懶人包:每題「技巧 + 作法」。各難度都收主流可遷移的;偏門/太難略過。難度 `~` 為 CF-equivalent。

- **[B. Predominant Frequency Division](https://codeforces.com/contest/2242/problem/B)** `貪心` `分類討論` `~1200`
  作法:值域 {1,2,3},切成 3 段,第 p 段要求「值 ≤ p 的個數 ≥ 值 > p 的個數」。**觀察:第 3 段永遠合法(無值 >3)**,故只需定第 1 段(count(1) ≥ count(2)+count(3))、第 2 段(count(1)+count(2) ≥ count(3))。用前綴和 +「掃描時維護前綴最小值」判可行(或 Kadane 式試所有中段起點)。

- **[C. Unstable Elements](https://codeforces.com/contest/2242/problem/C)** `數學` `~1400`
  作法:把陣列壓成連續相同值的 block,取長度多重集。**觀察:每次操作讓所有「存活」block 各長 +1**,故達到的總操作數 = 某 baseline `s2` + `s1`(存活 block 數)的倍數。由大到小枚舉存活門檻 i,累加 `s1`、`s2`;當 `k ≥ s2` 且 `(k−s2) % s1 == 0` 時答案 +1。O(n)。

- **[D. Two Digit Strings](https://codeforces.com/contest/2242/problem/D)** `dp` `不變量` `字串` `~1700`
  作法:操作 = 把相鄰兩位合併成「和 mod 10」。**觀察:digit sum mod 10 是不變量**。兩串總和不同即輸出 −1;否則要把兩串各切成最多段、且第 k 段「和 mod 10」對應相等 → 等價於在兩串的**前綴和(mod 10)陣列上做 LCS**(前綴和相等處可對齊),答案 = 共同切點數 − 1。把看似難的子段 DP 直接壓成 O(nm) 的前綴和 LCS。

- **[E. Product of Closures](https://codeforces.com/contest/2242/problem/E)** `位元` `構造` `貪心` `~2400`
  作法:一數的 closure = 其二進位表示無限重複、截到長度 n;兩數「乘積」= 兩週期串的 **bitwise AND**,要在 l ≤ x < y ≤ r 上最小化。**觀察(支配):把 x 的最低位 1 清掉後若仍 ≥ l,則原 x 被支配可捨去** → x 只需試「l 的前綴 + 在某個 l=0 位補 1 + 其餘補 0」的 O(log r) 個候選,y 同理,枚舉約 30×29 對取最小。可從暴力優化直接得解,省去繁瑣分類。

- **[F. Summer Vacation](https://codeforces.com/contest/2242/problem/F)** `資料結構` `根號` `~2500`
  作法:對初始序列 0..N−1,反向處理操作,每步在位置 a[i] 把序列切開重接(cut-paste),並查詢新序列的首元素。**觀察:純序列剪貼 → 用 implicit treap(split/merge)/持久化平衡樹即可**(參考解用持久化 treap + 週期重建控記憶體),或 sqrt 分塊。註:1.5s 時限下 O(n²) 向量化暴力也能過(社群熱議編譯器自動 SIMD)。

**不收**:~~A. Bigrams~~ → ad-hoc gimmick 判斷(`max(a)>2 或出現兩個 2` 即 YES),題意難可靠重建、對訓練無遷移價值。
